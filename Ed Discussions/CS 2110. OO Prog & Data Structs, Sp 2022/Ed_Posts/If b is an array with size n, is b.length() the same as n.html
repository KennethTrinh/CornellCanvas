In some Eclipse project, in some class, put the following. When done, you can delete it (or do this in jhhell): public static void m() { int n= 5; int[] b= new int[n]; System.out.println(n); System.out.println(b.length()); } What happens? What is wrong? Then look in this JavaHyperText pdf file, first page, to fix the error. https://www.cs.cornell.edu/courses/JavaAndDS/files/arrays.pdf When fixed, somehow call this method to see what it prints. My point here is to help YOU find answers to this yourself, and in a way that gets you use Eclipse and become more fluent with it.
