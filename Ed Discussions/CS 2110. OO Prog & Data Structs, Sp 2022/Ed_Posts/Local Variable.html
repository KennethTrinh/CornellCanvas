Yes, you can set the sum to zero before the for loop. You can do this: int sum=0; It will be your accumulator. Don't initialize variables in the loop body because the variables will be re-initialized every loop if the loop is run multiple times. In the loop body, you should only add to your sum(if you're trying to add things).
