The idea here is outlined in the class invariant. In this case we have a large array and we're only using the front portion of it to represent the schedule while the remaining portion of the array starting at index numC is unused. So by assigning the value of the first unused element of the array to a course that you're adding to the schedule and then adding one to numC, you're effectively adding c to the schedule.
