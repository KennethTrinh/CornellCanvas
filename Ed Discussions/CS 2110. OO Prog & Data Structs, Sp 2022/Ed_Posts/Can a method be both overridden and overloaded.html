Yes a method can both be overridden and overloaded. Here's an example: 
class Foo1 {
    int foo() {
        return 0;
    }
    
    int foo(int i) {
        return i + 3;
    }
}

class Foo2 extends Foo1 {
    int foo() {
        return 3;
    }
}
