1. Take a look back at TODO 8. It said to use a method related to scores. Read the body of that method. What does it do on its last line? With that in mind, consider a button instead of a property: your code does nothing to identify whether a button is clicked; rather, it reacts to that event upon being notified. It's the same with the property change. You don't write code to detect it; you write code to react to it. This is inversion of control -- and yeah, it's different than how we're used to coding.

2. You are trying to set the text in a JLabel. Take a look at the API page for JLabels, or, at the tutorial "How to use JLabels" linked from the first GUI lecture.
