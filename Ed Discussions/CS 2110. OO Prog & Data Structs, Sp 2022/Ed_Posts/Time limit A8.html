Your problem may be that you are timing it based on running with the GUI. Try it in Headless mode, running with only 1 seed. And fix code as shown below to print out the elapsed time for flee(). 1. Put this two items into class Pollack (you can leave them there): private long startTime= 0; // start time in milliseconds /** Return the time from start time till when this method is called, in milliseconds */ public long elapsedTime() { return System.currentTimeMillis() - startTime; } 2. Change your method flee() to put the first statement below as its first statement and the second statement below as its last statement. This will cause the elapsed time for flee to be printed. Be sure to comment out the println statement before you submit. startTime= System.currentTimeMillis(); System.out.println("Elapsed time: " + elapsedTime() / 1000.00 + " seconds");
