Author: Caleb Chin (student)

Vote Count: 0

Without the “this” modifier, it is not ambiguous which “gpa” is being referenced because of the bottom up rule. So just “gpa” will always reference the parameter by the bottom up rule. Using this.gpa will reference the field not the parameter also by the bottom up rule as “this” references the current object. So I guess in that sense, there is never ambiguity as to which one should be accessed because of the bottom up rule.
