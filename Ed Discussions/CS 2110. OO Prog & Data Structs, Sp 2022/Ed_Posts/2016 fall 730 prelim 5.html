For C.m(0)...

  • print 8

  • y is assigned to 0

  • print 7

  • if statement evaluates to true, so runtime exception is thrown

  • runtime exception is caught in catch branch so print 2

  • lastly print 1

    • So, you end up printing 8 7 2 1

    For C.m(1)...

  • print 8

  • evaluating expression to assign y divides by 0, since this is not in the try statement it cannot be caught

  • program crashes

    • So, you end up just printing 8
