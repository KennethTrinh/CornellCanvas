Why does the animal constructor need "this" in this.name = name. However in the rabbit constructor it's not this.height = h, but height = h? I thought it would be this.height = h.

----------- REPLIES -----------

Author: Vladia Trinh (staff)

Vote Count: 1

ENDORSED

this is necessary to distinguish the field name from the parameter name when they are called the same thing. But in the second case, the field is called height and the parameter is called h, so it is not necessary. Having this.height= h is correct, but we prefer to omit it to be more concise.
