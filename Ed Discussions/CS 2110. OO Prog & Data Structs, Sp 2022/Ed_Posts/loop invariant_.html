Yes, h can equal k. If h = k-1, the segment b[h..k-1] contains ONE value. If h = k, the segment b[h..k-1] contains ZERO values. The number of values in b[h..k] is (Follower - First) k - h. If you want the number of values in this segment to be 0, you need h = k.
