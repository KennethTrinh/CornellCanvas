Top one is a method from the Double class that checks if a Double has an equivalent value with another object. In this case the object is a Double with the same values so it returns true. In fact, you can read exactly what .equals does for the Double class: Bottom one is an operator that checks the location in memory of two objects. The two Doubles in your example are not the same object. They have two distinct locations in memory. So == is false.
