A char is represented by an int. You can cast, like this:

The value of ( char ) 65 is 'A' The value of ( int ) 'A' is 65

When arithmetic operators are used with characters, Java automatically coerces chars to ints:

'A'+1 evaluatesto 66 ( char )('A'+1) evaluatesto'B' 'A' < 'B' evaluates to true 'Z' +1 - 'A' evaluates to 26
