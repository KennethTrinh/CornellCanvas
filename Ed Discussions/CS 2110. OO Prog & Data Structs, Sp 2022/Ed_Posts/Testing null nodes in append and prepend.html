Both the append method and the prepend method do not specify whether the parameter 'v' can be null or not. Should we test cases when appending and pretending a null node? For example... dl.append(null); assertEquals("[null]", dl.toString());

If it is not specified, assume that you are allowed to assign the value of a particular node to be null. This doesn't mean the node itself is null, but rather the value v (the field) of the node is null.
