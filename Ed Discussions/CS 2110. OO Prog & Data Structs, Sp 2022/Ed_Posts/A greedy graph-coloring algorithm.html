Yes.

Not the particular implementation but you should be able to execute it yourself:

1. Color the nodes in some order.

2. When coloring a node, color it with the lowest color that is not the color of a neighbor.

This makes it a greedy algorithm: Try to use as few colors as possible by always choosing the lowest possible color.
