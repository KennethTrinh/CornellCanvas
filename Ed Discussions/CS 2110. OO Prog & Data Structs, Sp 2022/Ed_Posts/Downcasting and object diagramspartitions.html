Say class B is a subclass of class A, and we create an object of class A. In the object diagram, does the partition for class B exist when we first create the object of class A? Or does the lower partition for class B only appear if we do a downcast to class B?

A a = new A();

a = (B) a;

----------- REPLIES -----------

Author: Tomer Shamir (staff)

Vote Count: 1

If B is a subclass of A, creating an object of class A will not create a B partition since B extends A. Creating an object of class B will create an A partition.
