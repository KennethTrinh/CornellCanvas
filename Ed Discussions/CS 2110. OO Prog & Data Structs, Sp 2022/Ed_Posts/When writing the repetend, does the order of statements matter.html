Yes, the order of s and k matters. For example, let s = 0 and k = 1. If we had the order: s = s + k-1; k=k-1;, s would become 0 and k would become 0. If we reverse the order, and have k=k-1; s = s + k-1;, k becomes 0 and s becomes -1 as k has already been decremented in the previous statement. Each assignment statement is one after the other, so any changes to one variable will remain for the second statement.
