DO you know what a set is? Interface Set.

You know the operations on it?

Well, HashSet implements that interface., so you know the important operations that HashSet implements, those of interface Set. In addition, you should be familiar with the times these operations in the HashSet implementation. For example, inserting something into a HashSet is expected time O(1) but worst-case time O(n) --even if n is 2000.

In the same way, HashMap implements interface Map.
