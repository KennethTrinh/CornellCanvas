This is probably because you didn’t account for the case when there’s no head and tail. When you append a node and there is currently no head or tail, then both the head and tail should be that node you appended. Make sure to update your head and tail fields accordingly. Also to create a new node you must do

Node test = new Node(previous,val,next) (if i remember correctly at least), this is something you should double check with the constructor in the inner class Node
