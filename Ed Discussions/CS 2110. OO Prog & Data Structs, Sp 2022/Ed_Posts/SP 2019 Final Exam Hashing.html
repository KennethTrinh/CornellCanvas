a) x->10x works because the hash table has a fixed size 10. Because 10(1) mod 10 = 0, 10(3) mod 10 = 0, 10(7) mod 10 = 0, and 10(9) mod 10 = 0, every insertion results in a collision.

b) The size of the new rehashed array is twice the old array. For how to rehash, see #2356
