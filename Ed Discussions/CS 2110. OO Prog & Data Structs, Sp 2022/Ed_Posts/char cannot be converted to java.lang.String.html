You can convert char to String using Character.toString(char) or String.valueOf(char) . I'm not sure where you are getting this error from, I'm assuming its some method that expects a string argument like Integer.parseInt(string) . The reason that this doesn't work is just because the method was created to expect a string argument. If you wanted to, you could write a method or overload this method to take a char argument, and then do something like this: 

public static int parseInt(char c) {
    return Integer.parseInt(String.valueOf(c));
}
