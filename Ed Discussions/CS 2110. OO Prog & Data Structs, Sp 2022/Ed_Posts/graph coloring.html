We want to determine the smallest possible color. Since there are vs.length() neighbors, the smallest possible unused color is one of 0, 1, 2, 3, ... vs.length(). That's one of the first vs.length() + 1 colors. So, we want to make a list of the neighbors colors that are one of 0, 1, 2, 3, ... vs.length(). In making this list, we don't care about neighbors colors that are not in 0, 1, 2, 3 ..., vs.length().
