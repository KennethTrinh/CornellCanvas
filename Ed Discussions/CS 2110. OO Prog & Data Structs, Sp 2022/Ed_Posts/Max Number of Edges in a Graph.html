Suppose the graph has n nodes, numbered 0, 1, 2, ..., n-1.

Let it have:

Leaving node 0: n-1 edges to nodes with numbers in 1..n-1

Leaving node 1 (and not counted in the above): n-2 edges to nodes with numbers in 2..n-1

Leaving node 2 (and not counted in the above): n-3 edges to nodes with numbers in 2..n-1

...

Add up (n-1) + (n-2) + (n-3) ... + 1 + 0/
