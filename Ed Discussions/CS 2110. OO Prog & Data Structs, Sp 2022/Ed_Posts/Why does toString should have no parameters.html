When overriding the method toString(), why can't parameters be added? are we not defining a new function that takes in different types that uses the method name toString?

Author: Atulya Lohani (student)

You are not defining a new method. You are just overriding the toString() from Object which has no parameters. You can’t change the method header if you’re overriding, it has to be the same. toString() returns a string representation of the object so it doesn’t need to have parameters.
