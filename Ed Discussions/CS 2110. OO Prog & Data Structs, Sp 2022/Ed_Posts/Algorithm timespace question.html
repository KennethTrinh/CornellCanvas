Is time complexity the same thing as space?

For example, is an algorithm that has a constant time complexity the same thing as an algorithm that can be done in constant space?

----------- REPLIES -----------

Author: David Gries (admin)

Vote Count: 0

time complexity is about the time it takes to execute the algorithm.

space complexity is about the extra space the algorithm takes while executing.

They are different.

And our very own Prof. John Hopcroft proved in 1977 that an algorithm that runs in O(f(n)) time also runs in O(f(n)) space! Rough intuition: if the algorithm can use at most O(f(n)) time, then during that time it can access at most O(f(n)) space, but not more. Because to access more space, it would need more time!

If this sort of thing interests you, take CS 4810. Lots of neat results in that class about how hard it is to compute various algorithms.
