For part a you wrote compareTo using decreasing order. For part b, notice the invariant given for selectionSort says "inv: b[0..i-1] is sorted and b[0..i-1] <= b[i ..] (using compareTo)".

According to the invariant selectionSort should be sorting least to greatest with respect to what compareTo returns (which may be decreasing number of copies but selection sort is only concerned with what compareTo returns). In which case, keeping the invariant true, b[i] would switch with the min of b[i ..]
