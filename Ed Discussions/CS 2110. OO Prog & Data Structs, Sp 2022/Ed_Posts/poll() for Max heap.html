I am kind of confused about the first line in the while loop for the bubble down helper function for poll(). As I understand, the (h+1==n) checks if there exists a right node. If suppose there is no right node, then h+1>n and the first half of the conditional statement will be false. Author: David Gries (admin)

Vote Count: 0

Short circuit evaluation!

If h+1 == n, then the first term is true, so the whole condition is true, and the second term b[h] > b[h+1] is not evaluated.

Isn't that neat?
