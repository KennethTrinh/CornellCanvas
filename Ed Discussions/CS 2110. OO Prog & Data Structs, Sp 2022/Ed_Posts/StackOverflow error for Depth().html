StackOverflow means the stack is full, and you cannot use up any more space in the stack. This can happen from infinite recursion. Your base case could be a problem. Do you return 0 if human equals p? If the problem isn't the base case, do you have a return statement inside your for loop? Also, you need to return -1 if p is not in the tree or if p is null(you don't need a special if statement for this). You should have three distinct return statements for this method.
