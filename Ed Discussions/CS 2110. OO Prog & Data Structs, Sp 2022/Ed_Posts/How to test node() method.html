You can use the node.value() method to know the value of the node. To use this information effectively you want to append unique values, so let's say you have at node 5 the value "2110" you must make sure no other node in the list has the value of "2110" (this is something you should make sure as a tester when making your list) then you can use assertequals to make sure the value at node(5) is "2110", testing the node directly would be confusing (annoying) and unnecessary knowing this fact.
