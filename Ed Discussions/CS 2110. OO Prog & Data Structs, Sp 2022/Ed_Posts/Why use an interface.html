Whenever you implement an interface, the class implementing the interface is required to have an implementation for the interface methods. In other words, you can think of an interface as a framework for a few methods that some class must follow in order to implement said interface. For example, interface List has a few methods that must be implemented in order for a list to work properly and then classes like ArrayList and LinkedList are required to have implementations for the methods in interface List. This forces the programmer who is writing those classes to write an implementation for the basic methods that essentially make a list a list.
