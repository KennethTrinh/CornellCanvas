You are right. The return statement should be changed to obR.maxDepth == maxDepth. It is a primitive int and has no methods, let alone the equals() method.

For what it's worth, I'll correct the code you wrote as well:

Submarine obR = (Submarine) ob; if(obR.maxDepth == this.maxDepth && super.equals(ob)) return true;

The first line could lead to a ClassCastException because we don't know whether or not ob is actually a Submarine. All we know is that ob is just of Object type. Therefore, we call the parent class's equals() method first, which checks to make sure ob is of Submarine type and makes those other field checks.
