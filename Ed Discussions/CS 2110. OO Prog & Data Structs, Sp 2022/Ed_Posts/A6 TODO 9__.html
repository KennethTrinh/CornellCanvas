Author: Michael Clarkson (admin)

Conceptually there is no difference between PropertyChangeListeners and any other kinds of Listeners. For example, TODO 9 is right below TODO 3 in the code. In TODO 3 you added an ActionListener using an anonymous function. You do the same thing for a PropertyChangeListener in TODO 9. Just as the anonymous function in TODO 3 takes an argument of type ActionEvent, the anonymous function in TODO 9 can take an argument of type PropertyChangeEvent. TL;DR: use what you did for TODO 3 as a model, rather than something more complicated.
