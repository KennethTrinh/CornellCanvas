The shortest-path algorithm uses a HashMap. That HashMap is what is being talked about. Suppose you change the shortest-path algorithm so that it does NOT stop when it reaches end, so you don't need that second parameter.

Now, have the call shortest-path(exit), return the HashMap. It contains the shortest paths from exit to all nodes. Since this is an undirected graph, use as containing the shortest paths from all nodes to the exit.
