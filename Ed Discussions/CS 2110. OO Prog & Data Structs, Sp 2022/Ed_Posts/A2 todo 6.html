That means you're not using the return correctly. You're using return in a manner that is redundant so the eclipse formatting preferences just gets rid of it when you save. Rather than having an else block, just use the return keyword in your if statement. And then you can put everything that used to be in your else block, after the if statement. If the if statement body is run, then the method will return after printing 1. Otherwise, it will print 2x because the if statement body will be skipped. This gets rid of the necessity of having an else statement because you are using short-circuiting your code through the return.
