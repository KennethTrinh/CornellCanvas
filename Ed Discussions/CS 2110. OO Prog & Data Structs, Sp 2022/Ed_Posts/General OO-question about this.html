If the parameter is the same name as the field name you use this.field = parameter to differentiate it to java, because it searches from bottom up sees the first x as the parameter, so if you don't use keyword 'this' it doesn't know that you're referring to the field.

so let's say we had

public class C{

privant int x;

public void C(x){

this.x = x ;

//this.x refers to the private int field x

// x refers to the parameter x

} }
