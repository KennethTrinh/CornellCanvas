The value d[w] for any node w in the settled or frontier set is the length of the path (i.e. the sum of the weights on the edges of the path) from the start node to node w. Figure out how to calculate that. Suppose (node, Info Object) is in the HashMap. Use a Hashmap function, you can get a pointer to the Info Object. Then you can simply reference/change the fields directly.
