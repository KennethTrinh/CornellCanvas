Whether one uses add(val, priority) or add(val) depends on the heap that is being constructed.

If creating a heap where the values are also the priorities, then the class has method add(val).

If creating a heap in a class where the values and priorities are different. use the 2-par add method.

In the class used in this question, the values and priorities are different.

That first assignment to heap and loop following it is constructing a heap of the elements in s.
