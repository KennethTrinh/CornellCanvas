From: https://docs.oracle.com/javase/8/docs/api/java/awt/Button.html#addActionListener-java.awt.event.ActionListener-

" public void addActionListener( ActionListener l)

Adds the specified action listener to receive action events from this button. Action events occur when a user presses or releases the mouse over this button. If l is null, no exception is thrown and no action is performed." -- In this case, this (or IncrementButton) is l . IncrementButton is the ActionListener passed into addActionListener to receive events from this button (which is IncrementButton)
