Here h is to the right of the line not left which is why it's 0 not -1.

so b[0..h-1] must be empty so by our handy rule! we know

h-1 +1 - 0 = 0 so h = 0

b[h...m-1] must be empty, again:

m-1+1 -h = 0

m = h (we know h = 0, so both works)

b[m...t-1] is the whole array and we know m = 0 so b[0...t-1] is whole length of whole array

t-1+1 - 0 = b.length

t = b.length, we can also use visual inspection and see for b[0...t-1] to be whole array t must be b.length
