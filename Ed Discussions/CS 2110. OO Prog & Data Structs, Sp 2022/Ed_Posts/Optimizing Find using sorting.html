You can do it manually. Some people have put the nodes into a min-heap with priority being the distance to the Orb. But its easier to use some method that automatically uses some sort algorithm. Look at class NodeStatus. It implements comparable<NodeStatus>! And if you look at its method compareTo, you see that it compares two objects based on their distance to the orb. So you can make a collection of the neighbors and sort them using some Collections method.
