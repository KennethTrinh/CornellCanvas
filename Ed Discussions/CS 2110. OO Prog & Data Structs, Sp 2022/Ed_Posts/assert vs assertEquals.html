Author: Andrew Chen (staff)

Vote Count: 1

assert is used for assertions, so at the beginning of a function when you want to assert some preconditions (suppose I have a variable x of type int): assert x == 3; assertEquals() is a function used in your JUnit testing file. When you want confirm the returned value of a function is some value, you use it like this (suppose I have a function isEven() that takes in one int argument and returns a boolean telling me whether or not the argument was even) assertEquals(true, isEven(2)); Very different usages in very different contexts!
