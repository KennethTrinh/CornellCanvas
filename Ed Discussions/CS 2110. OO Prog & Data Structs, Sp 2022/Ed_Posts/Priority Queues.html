Look at JavaHyperText. Type "priority" into the Filter Field and look at the one pdf file there. The first thing the pdf file does is define priority queue, giving the values and the operations on them. That is a definition of a TYPE. We would call it an ABSTRACT DATA TYPE because there is no notion of implementation. Later, several implementations are given for this ADT.
