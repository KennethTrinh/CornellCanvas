I think the solution to this question is wrong. The tightest complexity should be $O(n)$ in my opinion. Please let me know if I miss anything.

The main issue of the current solution is that the complexity bound(O(n)) of inner loop is loose. So for any $n$, let's suppose $ 2^{k} \le n < 2^{k+1}$. Then, we have $k+1$ outer loops. For $i$-th outer loop(i starting from $0$), the complexity of corresponding inner loop is $2^{i}$.

So the total complexity is $\sum_{i=0}^{k}2^{i} = O(2^k) = O(n)$. This is the tighter bound I found.
