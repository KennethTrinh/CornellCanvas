Without seeing the code/error I cannot be sure this is the specific issue you are referring to however, TODO 9 should not be using an ActionEvent. The listener in TODO 9 is a PropertyChangeListener. Therefore the event should be a PropertyChangeEvent. (Note that every listener type has a matching event type, this is by convention)
