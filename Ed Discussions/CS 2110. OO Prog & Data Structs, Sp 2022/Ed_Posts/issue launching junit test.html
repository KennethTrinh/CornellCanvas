It's probably because the name for your test file is a2Test.java , whereas your class declaration is named A2Test . Note the different in capitalization with the first letter. Eclipse can be picky with file names - normally, you should always capitalize the first letter of a class in Java, and you want the file name to be named exactly the same as the class defined within it. Try right clicking on the a2Test.java file, and then clicking Refactor -> Rename, and then rename it to A2Test.java
