This probably means you are returning inside an if statement and Java's compiler can't guarantee that the if statement will be entered so if none of the if statements are ever entered then then there is nothing returned is what this error message means. There are two ways to fix this, either break out of the loop and return what you'd normally return (when the if statement is satisfied), the other option is to return a random Node at the very end of the method that will never be reached (assuming your if statements are always guaranteed to be entered)
