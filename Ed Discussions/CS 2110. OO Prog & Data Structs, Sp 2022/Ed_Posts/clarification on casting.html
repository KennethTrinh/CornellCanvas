This is just for the compiler to know that if you do S secret = (S) ob; then you can use methods in class S using the secret variable without the compiler raising an error, let's assume method .cry() is in Class S. if you did ob.cry() you'd get a compiler error (compile time reference rule) Even though the dynamic type is an instance of Class S. but if you did ((S)ob).cry() you'd not get an error.
