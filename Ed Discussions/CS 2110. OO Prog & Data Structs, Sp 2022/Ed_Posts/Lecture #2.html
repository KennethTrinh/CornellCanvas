When you execute at the assignment x= k; use the inside-out rule to determine the declaration associated with x and with k. In both cases, the inside-out rule tells you that the variable (x, k) refers to the parameter. Thus, this assignment just stores the value in parameter k into parameter x. Thus, after this assignment, both parameters contain 3. Now, execute the second statement, c.y= x; . The inside-out rule tells you that x refers to parameter x, whose value is 3. The, the right hand side evaluates to 3. y refers to field y of c, since this method call is c.m(3, 4). The assignment changes c.y to 3.
