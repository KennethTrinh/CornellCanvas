Author: Tomer Shamir (staff)

Vote Count: 1

In the structure of this assignment, we want the advisor to be a PhD because PhD has information such as the number of advisees. As a result, we want to include this information in the object that is "linked" to a PhD as its advisor instead of just the name of the advisor. By using a PhD object we can include not just the name but all the other information we get from the rest of the PhD fields.
