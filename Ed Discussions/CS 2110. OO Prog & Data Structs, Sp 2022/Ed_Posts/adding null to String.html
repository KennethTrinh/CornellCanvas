Why would it be an error? You are not calling a method on null, you are catenating a string and null is converted to a string. I searched this on google: The concat() method only accepts a String object as an argument, while the + operator will accept anything. If an object is passed to the + operator, it gets converted to a string by the toString() method of that object. If a null reference variable is passed to the + operator, it gets converted to string "null" .
