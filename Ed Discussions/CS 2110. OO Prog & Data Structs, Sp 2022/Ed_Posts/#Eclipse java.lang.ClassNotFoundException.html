Problem: Error: Could not find or load main class <classname> Caused by: java.lang.ClassNotFoundException: <classname> Solution: Right click on your project and go to Build Path -> Configure Build Path. Go to the Libraries tab. Once you're in the library tab, click on "JRE System Library" and then click the edit button. Then, in the Execution Environment dropdown menu, make sure that it is jdk, not jre. If it is jre, click on the dropdown menu and find jdk for Java 11 and select it. Then click Finish. If you can't find jdk as an option, you either have downloaded jre and not jdk or the jdk may not be in the path. In the edit menu, try clicking on the Installed JRE's button and click Add. Next select Standard VM and click Next. Click the Directory Button and look for your jdk 11. It may either be in Library/Java/JavaVirtualMachines or Library/AdoptOpenJDK for Mac or C:/Program Files/Java or C:/Program Files/AdoptOpenJDK for Windows. The folder you need to find inside the directory should start with "jdk-11." Note: you may still get the error after doing this. If you do, restart eclipse.
