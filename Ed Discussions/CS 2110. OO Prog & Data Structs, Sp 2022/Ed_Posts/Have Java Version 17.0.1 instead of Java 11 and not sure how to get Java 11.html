It's okay to have a new version of Java on your computer. However, you must change the compiler compliance level in Eclipse. To do this, you have to go to Eclipse's preferences, click the arrow next to java in the sidebar to pull up a submenu, click on compiler, and change the compiler compliance level to 11. You should closely follow the instructions on the canvas page.
