Whenever your code does something weird like divide by 0 for example, Java create an object and throws it. Throwable is the super class of the Error and Exception classes. So, when Java throws an object, you can write code that catches the object. In this case, if a Throwable object is caught you print "F". The point of this is so that your program doesn't crash when weird exceptions happen. After the Throwable object is caught, you just move on to the next line of code, which is to return -2. For more info, search "throwable" in Java Hypertext.
