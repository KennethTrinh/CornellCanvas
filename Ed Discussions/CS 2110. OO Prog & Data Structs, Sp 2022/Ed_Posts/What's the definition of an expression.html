An expression is not a line of code. Think of it like a mathematical expression that can be evaluated to some value except not all expressions are numerical expressions. For example, "hello" + "world" is an expression and 'c' + 5 is also an expression and 56 + x is also an expression where x is a variable of type int. And in the assignment statement x= y; the righthand side y is an expression.
