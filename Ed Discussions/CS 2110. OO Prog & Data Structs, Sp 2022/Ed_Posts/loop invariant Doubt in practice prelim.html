b[e…h-1] is the unknown region, we want that as the whole array initially, we see so h-1 = f so h = f+1, we want b[h…k] to be an empty region so follower - first rule: k + 1 - h = 0 so k = h-1, we can leave it like this since we’ve initialized h, or plug in for h, h = f+1 so k = f + 1 -1 so k = f, We want to continue looping while the segment b[e..h-1] still has elements so h -1 + 1 - e != 0 in other words h != e
