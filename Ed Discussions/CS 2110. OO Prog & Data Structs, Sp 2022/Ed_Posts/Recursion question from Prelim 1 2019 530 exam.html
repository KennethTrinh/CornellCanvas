Sure, so F(3) doesn’t enter our base case so instead goes into the F(n-1) call, which would call F(2) which doesn’t go into any of the base cases, so this calls F(n-1) =  F(1) which does enter a base case, then the second part of the return statement is executed for the F(2) call which is F(n-2) = F(0) which enters a base case, then we go back to F(3) and do the call on F(n-2) = F(1) which enters a base case so we’re done.
