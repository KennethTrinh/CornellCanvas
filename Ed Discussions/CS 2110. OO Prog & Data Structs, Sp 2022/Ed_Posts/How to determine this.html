We want the catalogs ordered in decreasing inventory size, meaning bigger ones come first. The spec says that we want to return a positive int if this Catalog comes after ob. This means return a positive int if this Catalog is smaller than ob. You accomplish this with ob_size - size because if size < ob_size then we return a positive int.
