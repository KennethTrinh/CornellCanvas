A non-static method aka instance method is defined in JavaHyperText. It is a method that is part of every object of a class. A static method is also defined in JavaHyperText. It is a method that exists in only one place, as part of the class not any objects.

With that in mind, study the error message: what is it saying? what is it telling you about how Java understands the code you've written?
