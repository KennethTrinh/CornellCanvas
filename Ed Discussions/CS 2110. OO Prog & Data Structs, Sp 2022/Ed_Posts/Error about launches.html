Search up “method main” in JavaHyperText for more info, here’s some notes: I think the code you’re referring to is being run in JShell. JShell is not an IDE. It's important not to get confused between JShell and programming in an IDE, such as Eclipse. JShell lets us run code snippets without creating a java class and main method. But a proper Java application (written in an IDE) must have the main method, since that’s where all the execution happens (you’ll have to check the box when creating the class to add it, or yes, add it manually if you didn’t)
