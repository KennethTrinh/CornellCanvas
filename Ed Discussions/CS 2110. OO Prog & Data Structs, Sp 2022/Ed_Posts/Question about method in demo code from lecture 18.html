I thought foreach loops can only be used if Iterable is implemented. How does this method in the lecture demo work then?

public boolean hasVertex(VL v) {

for (Vertex u : vertices) {

if (u.label.equals(v)) return true;

}

return false;

}

Adjacency Lists are represented by the List interface, which are implemented by ArrayLists or LinkedLists, as seen in lecture. I believe these classes are automatically Iterable, so you don't need to manually implement Iterable/Iterator every time you want to use a for-each loop. You only need to do it yourself when it's not one of the classes imported from Java that do this already.
