In this case, the invariant says that v is the minimum value of the array segment b[k..n]. If you initialize k to 0 and v = b[0], you are saying that b[0] is the minimum value in array segment b[0..n]

Author: Tomer Shamir (staff)

The point of loop invariants is that this invariant says something that will always be true at the beginning and end of each loop iteration. In this case, that invariant is that v = the minimum value of b[k..n]. With this invariant, initializing k=0 and v=b[0] violates that loop invariant for the reason I explained above. We need to find an initialization that doesn't violate the invariant
