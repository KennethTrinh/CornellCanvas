Sure, we first draw -5 in the heap, then we add 10, -5 / 10 then bubble up 10, so 10 / -5 we now add 3, 10 / \ -5 3 (no bubbling up required in this max heap) we now add - 2 10 / \ -5 3 / -2 we see -2 > -5 so we have to bubble up 10 / \ -2 3 / -5 we now add 27 10 / \ -2 3 / \ -5 27 27 > -2 so we have to bubble up 10 / \ 27 3 / \ -5 -2 27 >10 so we have to bubble up 27 / \ 10 3 / \ -5 -2 We now add 13 27 / \ 10 3 / \ / -5 -2 13 13 > 3 so we have to bubble up 27 / \ 10 13 / \ / -5 -2 3 We don't need to double up for this anymore, we now add 84 27 / \ 10 13 / \ / \ -5 -2 3 84 84 > 13 so we have to bubble up 27 / \ 10 84 / \ / \ -5 -2 3 13 84 > 27 so we have to bubble up 84 / \ 10 27 / \ / \ -5 -2 3 13 (sorry if this is long but here you go)
