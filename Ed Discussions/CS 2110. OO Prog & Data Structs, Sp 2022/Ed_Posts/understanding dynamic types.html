Author: David Gries (admin)

Yes, the dynamic type is the name of the lowest partition.

No, an object doesn't point anywhere.

Suppose you have. an. of static type Animal, ca of static type Cat, and siam of static type SiameseCat.

Suppose they all contain a pointer to the same object, which has partitions for Object, Animal, Cat, and SiameseCat. That pointer points at the OBJECT, not at any particular partition.
