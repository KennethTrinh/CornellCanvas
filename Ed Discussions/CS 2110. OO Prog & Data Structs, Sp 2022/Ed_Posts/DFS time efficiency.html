for time required for DFS, my understanding is as follows:

1. Mark each node = O(V)

2. Recursive call on each unvisited vertex = O(V)

3. Finding each edge will depend on whether if its an adj list or matrix:

  • Adj list: O(E)

  • Adj Matrix O(V^2)

    • 4. Total:

  • Adj List = O(V) + O(V) + O(E) = O(2V+E), which is in O(V+E)

  • Adj Matrix = O(V) + O(V) + O(V^2) = O(2V+V^2) which is in O(V^2) as V goes to infinity (we only cares about V^2)
