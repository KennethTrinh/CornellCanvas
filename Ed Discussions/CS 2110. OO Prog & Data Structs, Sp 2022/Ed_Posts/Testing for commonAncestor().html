If the CovidRoute for child1 or CovidRoute for child2 is null then you know one of the children doesn't exist in the tree. So your commonAncestor method would return a null in that case since the two humans aren't in the same tree. All you have to do to test this is just make a CovidTree, and insert some humans. Then make a variable and store the return of CommonAncestor(human1,human2) in a variable. One of the humans passed in shouldn't be in the CovidTree since you want to get a null return. Then you just do assertEquals(null,variable that you created). Also, you can use makeTree1() that is provided, which lets you create a tree easily.
