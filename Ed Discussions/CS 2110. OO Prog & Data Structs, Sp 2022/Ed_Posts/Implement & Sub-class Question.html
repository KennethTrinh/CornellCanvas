Author: Bahaa Kotb (student)

Vote Count: 0

Because the parent class implements the Valuable interface we are guaranteed that it has a getvalue method therefore the subclass doesn’t have to have it.

If the parent class didn’t implement the Valuable interface then the subclass MUST implement the getvalue method
