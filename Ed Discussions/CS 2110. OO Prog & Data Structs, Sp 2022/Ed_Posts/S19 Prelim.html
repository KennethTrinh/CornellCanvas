Let's go through it, first we call M(2) which enters the return n - F(M(n-1)) call

that will first call M(n-1) which is M(1) and same concept, it'll enter M(0) then M(0) enters the base case and returns 0, so we go back down the stack to the M(1) call and then do F(0), that then enters base case and returns 1 then 1-1 =0 , we then go back to the M(2) call which would run F(0) since the return statement of the M(1) call returns 0.

This is not really easy to explain through writing though and I recommend drawing the call stack in this case.
