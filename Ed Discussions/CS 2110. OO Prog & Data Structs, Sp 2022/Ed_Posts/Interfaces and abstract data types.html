Consider an abstract type LLList. Its values are lists of integers. It has two operations:

append an element remove the last element that was appended

Below is the interface. The two declarations give the syntax of the operations. What they do is given by the specifications. 

interface LLList {

    /** append k to the LLList */
    public abstract void append(int k);

   /** remove the last element that was appended */
   public abstract void remove();

}
