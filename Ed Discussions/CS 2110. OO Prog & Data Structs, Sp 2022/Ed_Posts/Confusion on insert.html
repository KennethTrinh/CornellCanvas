1. The specification lists the conditions under which an IllegalArgumentException should be thrown.

That bolded throws clause is how one can indicate that the method throws an exception. Please look up "throws" in JavaHyperText for a full explanation.

2. It says: "Given p a new child, a node with Human c. Return the newly created node."

Well, create a new node and make it a child of p. Look at the fields of the class. Where are the children? As you create the new node, save a pointer to it; that is what is to be returned.
