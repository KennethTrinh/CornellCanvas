When the problem statement says "a random function $f$ on $X$ can be identified with an element $\vec{f} \in \mathbb{R}^n$, does that mean that we map each $i \in X$ to $e_i$, the $i$th standard basis vector in $\mathbb{R}^n$? Or does it mean that each component of $\vec{f}$ is a random element of $X$?

Author: Luke Serafin (staff)

Vote Count: 1

Let $f: X \rightarrow \mathbb R$ be a random variable. If we fix an enumeration $\{x_1,x_2,\ldots,x_n\}$ of the elements of $X$, we can identify $f$ with the vector $\begin{pmatrix} f(x_1) \\ f(x_2) \\ \vdots \\ f(x_n)\end{pmatrix}$.
