Yes, it would still hold. You can replace that line by "$DF(\mathbf{a})$ has rank $m$". So why did I write "$DF(\mathbf{x})$ has rank $m$ for every $\mathbf{x}\in M\cap V$"? If $DF(\mathbf{a})$ has rank $m$, then $DF(\mathbf{x})$ has rank $m$ for all $\mathbf{x}$ in some neighbourhood $V_0$ of $\mathbf{a}$ (which may be strictly smaller than $V$). This is true because $DF(\mathbf{x})$ depends continuously on $\mathbf{x}$ (as $F$ is assumed to be $C^1$). If $DF(\mathbf{x})$ has rank $m$ for all $\mathbf{x}$ in $M\cap V$, then you know that all of $M\cap V$ is a manifold, not just $M\cap V_0$.
