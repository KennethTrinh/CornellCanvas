The problem is corrected to just ask for local minima now (not strict), but I wanted to note that $C^1$ is not enough to guarantee strict local minima, and indeed $C^\infty$ is still not enough. There are $C^\infty$ functions with compact support (i.e. zero off a compact set) which have unique maxima, and if we string three of these together with an interval outside the support between each pair we get a counterexample similar to your continuous-but-not-differentiable counterexample. Requiring that $f$ be analytic, i.e. representable by a power series with infinite radius of convergence, is enough to guarantee strict local minima because an analytic function which is constant on some interval is constant everywhere. That proof should be accessible via the methods of this class, but of course it is an exercise purely for your own enrichment if you decide to pursue it.
