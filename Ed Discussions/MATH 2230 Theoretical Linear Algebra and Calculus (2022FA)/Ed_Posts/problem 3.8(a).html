The function $\phi_1$ is a function of a pair $(\mathbf{a},\lambda)$. If you want to differentiate $\phi_1$ using the definition of total derivative, you have to write $$\phi_1(\mathbf{a}+\mathbf{h},\lambda+\delta)=\phi_1(\mathbf{a},\lambda)+L(\mathbf{h},\delta)+R(\mathbf{h},\delta),$$ where $L$ is a linear map and the remainder term $R(\mathbf{h},\delta)$ is of order $$o\bigl(|\mathbf{h},\delta)|\bigr)=o\bigl(\sqrt{|\mathbf{h}|^2+|\delta|^2}\bigr).$$
