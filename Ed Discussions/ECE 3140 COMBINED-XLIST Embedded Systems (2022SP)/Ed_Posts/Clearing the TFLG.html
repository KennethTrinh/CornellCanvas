If we set the TFLG to 1, which means to clear it, we enable the interrupt service once the time has run out. However I am unsure of where the timer gets resetted? Does setting the TFLG also restarts the timer by any chance?

Author: Harris Miller (None)

Vote Count: 3

Once the PIT hits zero (times out), the PIT stops and generates an interrupt. When an interrupt is cleared, the PIT will begin counting again starting at the value stored in the "Load Value" register.
