I know this response is late, but there are two necessary malloc() calls in this assignment - one that you will write and one from process_stack_init() . Therefore, there should be two free() calls, one in process_stack_free() and one you should write. If everything you've ever malloc'd is still in the heap, it could be that you are not calling free() properly, or you're doing it in a piece of code that never executes, or you're missing either your free() or process_stack_free() . If you've already submitted and are still worried about this, don't worry, we don't extensively test for memory leaks ;)
