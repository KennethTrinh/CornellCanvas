Author: Naomi Koh (staff)

Vote Count: 1

This special return value is used to denote that the ISR is being executed/an exception has occurred. Once the ISR is complete, it will execute BX LR which will pop the saved registers from the stack and return to the main program. It's basically an automatic mechanism that assists the exception return.
