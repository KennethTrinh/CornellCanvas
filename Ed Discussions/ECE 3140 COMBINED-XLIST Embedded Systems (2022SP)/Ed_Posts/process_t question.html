The following is found in 3140_concur.h : 
struct process_state;
typedef struct process_state process_t;
   /* opaque definition of process type; you must provide this
      implementation.
            
            To do this, write code similar to the following in your process.c:
            
            struct process_state {
                int var1;
                struct process_state* some_pointer;
            };
   */
As stated, you will be implementing the struct process_state in process.c . The fields of process_state are up to do you decide based on several factors: the implementation of your linked list, the process stack, and anything else that you may find helpful to store alongside each process.
