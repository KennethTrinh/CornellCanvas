& means "address of" so if you had some variable x, the expression &x would be the address of x. if you have a pointer and want to turn it into a double pointer, you would put & before the name of the pointer. for example, y is a pointer and for some function c you need to input a pointer to y (thus a double pointer) you could do c(&y)
