Here is the Canvas page from 2021:

-----------------

For the final project we ask you to implement an embedded system of your choice.

The logistics for this final portion of the class will be somewhat different. We won't have any new (required) material or quizzes so you can focus on the project and your other classes. I will still post lectures about some extra topics that I would like to cover, or specific questions that you might have and want to hear more about.

The main output will be a webpage and a short video.

The grading breakdown

  • 10% Proposal and Team from

  • 30% Webpage (%10 check in, %20 final))

  • 20% Video

  • 20% Code

  • 15% Your testing scheme

  • 5% Submitting your vote for the final video competition

    • Once you submit your team form we will create a github repo for your with an associated github page that you can edit. To manage the different pages we ask that you use github to host your webpage instead of some external service.

    Deadlines

    Proposal form: 5/6/21 Webpage checking: 5/14/21 Final Submission of code + Webpage: 5/23/21 Final video competition votes: 5/25/21

    Project Parameters

    You have to use the board from class in some significant way, but you can network it or interface it to other devices or boards. For the final competition, there will be two categories: projects without any additional hardware, and projects that use additional components. That way you have a fair shot at winning even if you don't have an electronics shop at home or access to fancy components. MCUXpresso has code examples for most of it's peripheral devices and you are welcome to use them in your project. For example, if you want to use your computer figuring out how to use the serial port over USB might be helpful.

    We will try to give you quick feedback on your project proposal and indicate whether or not it fits the project parameters and if we foresee any implementation issues. We'll also try to give you pointers on how to go about some technical issues we foresee. If you have a specific question on how feasible something is, you can put it in your project proposal form. If you submit the form early, we will start giving feedback before the due date.

    Good luck!
