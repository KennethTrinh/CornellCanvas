I did it this way (not sure if it's correct though):

First move n - 1 to the input register, call the fib function, then save the result in a register like R4 that is guaranteed to be maintained properly. Then store n - 2 to the input register, call the fib function again, and add the result to the value in R4 to obtain the final answer :)
