extern process_t* current_process; 
extern process_t* process_queue;

The extern keyword allows the variables to be declared such that the compiler knows of its existence, but memory is not actually allocated for them. You will still need to define the variables in process.c .

The following is an example of using extern :

lib.h 

extern int global_var;

main.c 

#include "lib.h"
int global_var;

int main(void) {
    global_var = 1;
}
