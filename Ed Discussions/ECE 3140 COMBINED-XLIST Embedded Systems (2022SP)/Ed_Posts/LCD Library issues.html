I saw another problem like this with a group earlier. There are some files that have the same name between demos but have different functionality within the methods (believe the files in the board directory are an example of this). If your project only needs to use the LCD I recommend just putting your source code into that sdk demo. Otherwise you'll need to make sure that you add any differences between the files as well.
