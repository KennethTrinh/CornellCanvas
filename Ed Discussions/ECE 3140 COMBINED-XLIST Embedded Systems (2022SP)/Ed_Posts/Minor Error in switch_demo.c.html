PORTC->PCR[SWITCH_1_PIN] |= PORT_PCR_IRQC(0b1011); // Set up the IRQC to interrupt when logic 0 was changed to: PORTC->PCR[SWITCH_1_PIN] |= PORT_PCR_IRQC(0b1011); // Set up the IRQC to interrupt on either edge (i.e. from high to low or low to high) Setting IRQC to 1011 will set the interrupt to interrupt on either edge, whereas setting it to 1000 will interrupt when the logic is zero. The comment was changed to reflect the actual behavior as coded. As a reminder, you can look at page 194 of the KL46 sub-family reference manual to see the other IRQC options to choose the behavior you want for your project
