Author: Harris Miller (None)

Vote Count: 0

Just like how writing to a single bit in PCOR or PSOR then clears or set the corresponding bit, writing to a single bit in PTOR will always toggle the corresponding bit. You do not need to try and read from it and then decide if you need to set or clear. The read value of the entire PTOR register is always 0.
