One option suggested by Aaron was to generate a seed by reading an analog value from one of the GPIO pins that is not connected to anything. As the pin will be floating random EMF noise will be read, giving you a random seed for your generator. Some of this is mentioned in the GPIO discussion where we read the input from a switch connected to a GPIO pin.

Author: Nils Napp (admin)

Another way to get a random number would be to start a timer at boot time and then use the time until a user first presses a button. That way you don't need to mess with analog reads.

A single read from a digital pin will probably not be enough to randomize the seed sufficiently.
