One possibility is to create a new project and new repository, clean out the C files from source , and then clone the repo into the source folder. While this will not retain any project settings like excluded files, it should still be a whole lot better than pushing / pulling all the generated output and debug files and saves the hassle of .gitignore on everything besides your C code.
