Title: Relationship between Q0 and overdamped/underdamped cases

Is there a relationship between the value of Q0 for an oscillator and the overdamped / underdamped / critically damped cases? i.e., can we tell if an oscillator will have a high Q0 based on the value of b^2 - 4mk?

----------- REPLIES -----------

Author: Dan Ralph (admin)

Vote Count: 0

Sort of. In the underdamped case, one can write $Q_0 = (1/b) \sqrt{mk}$, which means that $(b^2 - 4mk)/b^2 =1-4Q_0^2$. So if $(b^2 - 4mk)/b^2$ has a large negative value than $Q_0$ is large.
