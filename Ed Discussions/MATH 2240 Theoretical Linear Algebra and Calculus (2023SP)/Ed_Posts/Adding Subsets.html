\text {Is it possible to add two subsets } A\subset \mathbb{R}^n \text { and } B\subset \mathbb{R}^m \\ \text{ and say that the result is some subset in } \mathbb{R}^{n+m} \text{?}

----------- REPLIES -----------

Author: Reyer Sjamaar (admin)

Vote Count: 0

It’s not called “adding” but “Cartesian product”: for arbitrary sets $A$ and $B$ the Cartesian product $A\times B$ is defined as the set of all pairs $(a,b)$ with $a\in A$ and $b\in B$.  If $a$ is an $n$-vector and $b$ an $m$-vector, we can think of $(a,b)$ as the $n+m$-vector $\bigl(\begin{smallmatrix}a\\b\end{smallmatrix}\bigr)$.
