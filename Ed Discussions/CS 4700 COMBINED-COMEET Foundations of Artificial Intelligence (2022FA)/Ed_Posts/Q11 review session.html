Author: Laura Greenstreet (admin)

Yes, you likely wouldn't want to use a shortcut feature as while it will improve your performance on the biased test/training data your model will likely not perform well for its intended application. The review question isn't focused on whether you would want to use the feature but whether it would perform well. It is quite hard to deal with shortcut features/biased data. Even detecting that the model is using a shortcut can be challenging for some models like neural networks where how the model is using the input values is not easily interpretable.
