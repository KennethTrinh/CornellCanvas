The power set of $A$ always exists, but if $A = \phi$ the power set is just $\{ \phi \}$ and $\mathcal{P} _ 0 ( A )$ is $\phi$, so a choice function doesn't exist. The Axiom of Choice actually says a choice function exists for every nonempty set --- I'll fix the problem so it says that.
