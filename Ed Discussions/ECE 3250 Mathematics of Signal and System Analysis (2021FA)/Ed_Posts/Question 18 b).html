Any constant input $x ( n ) = c _ o$ is of the form $x ( n ) = c _ o z _ o ^ n$ for $z _ o = 1$, An input of the form $c _ o z _ o ^ n$ works for the system precisely when $z _ o$ is in the ROC for $H ( z )$, and the output signal is then $c _ o H ( z _ o ) z _ o ^ n$. Plug in 1 for $z _ o$ and you get the answer to this problem.
