Would the sup and inf have to fall into a the rational numbers set for a sequence of rational numbers.

----------- REPLIES -----------

Author: David Delchamps (admin)

Vote Count: 0

No. $\sup ( A ) \in A$ if and only if $\max ( A )$ exists. Like, $\sup ( ( - 3 , 3 ) ) = 3 \notin ( - 3 , 3 )$.
