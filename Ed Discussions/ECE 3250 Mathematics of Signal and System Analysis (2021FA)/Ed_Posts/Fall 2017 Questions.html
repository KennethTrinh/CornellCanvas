9(i): $x$ bandlimited implies $x$ has a Fourier transform (wouldn't make sense otherwise). 10(ii): The I/O mapping is in the standard LTI form of $S ( x ) = h _ 0 * x +$ a shift term. $h _ 0 ( t ) = e ^ {- t} u ( t + 1 )$. 11(iii): The output will have terms at exactly the same frequencies as the input because $\widehat{H} ( \Omega )$ is never zero, so the output has the same fundamental period as the input. 14(iv): Write out $A$ in SVD form and take $A ^ H$ and this is what you get. 19(b): You have to make sure each signal gets pulse-sampled by the switch faster than its Nyquist rate, so the signal with the wider bandwidth needs to get sampled twice every cycle.
