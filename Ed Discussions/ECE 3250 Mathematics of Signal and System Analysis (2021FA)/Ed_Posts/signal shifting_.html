$S ( x ) ( t ) = x ( 3 t - 1 )$ for any signal $x$. When $x = {\rm Shift} _ 1 ( u )$, $S ( x ) ( t ) = {\rm Shift} _ 1 ( u ) ( 3 t - 1)$ ${\rm Shiftt} _ 1 ( u ) ( {\rm anything} ) = u ( {\rm anything} - 1)$, so ${\rm Shift} _ 1 ( u ) ( 3 t - 1 ) = u ( 3 t - 2 )$.
