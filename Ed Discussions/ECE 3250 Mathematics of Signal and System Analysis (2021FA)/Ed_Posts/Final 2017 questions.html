1) A mapping from the naturals to set A can be surjective but every mapping doesn't necessarily have to be. For example f(x) = 2 goes from all the naturals to just one value (not surjective). A mapping doesn't have to "hit" every value in the set A to be a mapping.

2) The question specifies the set of all irrational numbers between -1 and 1 which are uncountable. Rationals and Integers are countably infinite, irrationals are infinite just not countable.
