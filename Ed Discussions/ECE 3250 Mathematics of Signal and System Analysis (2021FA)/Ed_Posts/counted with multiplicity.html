in class we said that the rank of an Hermitian matrix Q is the number of nonzero eigenvalues of Q "counted with multiplicity"

Actually just the opposite. Like for example the $( n \times n )$ identity matrix has only one distinct eigenvalue, namely 1, bot that eigenvalue has multiplicity $n$, and the identity matrix has rank $n$. It has "$n$ nonzero eigenvalues, counted with multiplicity."
