What does cardinality of the instance space mean?

----------- REPLIES -----------

Author: Victor Butoi (staff)

Vote Count: 1

This means how many unique instances, of a set of features, can possibly exist; given how many different possibilities there are for each feature. Each X instances is a vector of features after all, where each of these feature manifests a possibility of that kind of data.
