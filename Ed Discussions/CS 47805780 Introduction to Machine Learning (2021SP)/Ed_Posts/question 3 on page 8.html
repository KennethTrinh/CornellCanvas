I interpreted this to mean directly computing the classifier output as

y\ =\ w'\cdot x'

since for a regression problem, y is real-valued. Since it the support is not countable, using a piecewise classifier ( h(x) in the binary classification problem) doesn't really make sense. The label is a real value and can be computed directly rather than forced into a set of discrete values like {-1,1} in the binary classification. problem.
