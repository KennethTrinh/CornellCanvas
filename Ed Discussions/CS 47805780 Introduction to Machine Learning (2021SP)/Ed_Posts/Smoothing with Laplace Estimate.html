Let $X_{i}$ and $Y$ be binary. Imagine a sample with only one instance in it: $(X_{i} = 0, Y=1)$. If you measured $|X_{i}|$ based on the sample set, you would get that $|X_{i}| = 1$ (i.e. $X_{i}$ can only take on one value). Then you would have: $$\hat{P}(X_{i} = 0 | Y = 1) = \frac{0 + 1}{1 + 1} = \frac{1}{2}$$ and $$\hat{P}(X_{i} = 1 | Y = 1) = \frac{1 + 1}{1 + 1} = 1$$ But then, $$\hat{P}(X_{i} = 1 | Y = 1) + \hat{P}(X_{i} = 0 | Y = 1) = \frac{3}{2} \neq 1$$ When you use the theoretical value of $|X_{i}|$ you get instead: $$\hat{P}(X_{i} = 0 | Y = 1) = \frac{0 + 1}{1 + 2} = \frac{1}{3}$$ and $$\hat{P}(X_{i} = 1 | Y = 1) = \frac{1 + 1}{1 + 2} = \frac{2}{3}$$ In summary, using the theoretical $|X_{i}|$ guarantees that $\hat{P}$ stays a valid probability distribution.
