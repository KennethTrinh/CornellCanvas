The hypothesis class in this example is not equal in size to the full set of possible mappings from the instance space $X$ to the output space $Y$. In this question, the full set of hypotheses is restricted to linear classifiers with weights $w_{i} \in \{-2, -1, 0, 1, 2\}$. When calculating $|H|$ you should use this set and not the full set of possible mappings.
