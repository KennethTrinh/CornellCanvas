1. It's asking for the number of values you need to store to calculate any classification; i.e. however many parameters you need to completely specify the naive bayes model.

2. A parameter is anything used in the naive bayes model, i.e. NB doesn't directly have a P(Y=1|X=(1,1,1)), but rather calculates it using information it has from the training data.

2. The "final event" is talking about the one unknown event described in the statement "knowing the probability of all events except one event."
