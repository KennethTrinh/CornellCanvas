In lecture, it was mentioned that the HMM allows us to only have "a squared number of parameters in the number of part-of-speech tags" -- I don't quite understand how we got to this conclusion? How do we know that the number of parameters is squared? What are the parameters in this case (i.e. are the parameters the part-of-speech tags?) ?

It was mentioned that HMM takes an exponential-size problem and brings it down to a polynomial number of parameters -- how do we know that HMM gives a polynomial number of parameters?

Because you need the transition probabilities from $y_{i-1}$ to $y_{i}$ for all possible values of $Y$. Since $Y$ is the set of POS tags, you have a total of $|Y \times Y| = |Y|^{2}$ parameters (where each parameter is an estimate for $P(y_{i}|y_{i-1})$).

Without HMM, you would need to calculate the joint probability $P(Y = (y_{1}, ... , y_{l}))$ for all possible sequences which is exponential (would be order $|Y|^{l}$ where $l$ = length of sequence).
