Err(S), the size of the minority label is 2 Err(S|A) = 1 + 0, because when you split on A the size of the minority labels is 1 for A=0 and 0 for A=1 Err(S|B) = 2 Err(S|C) = 2 so the max Err(S) - Err(S| Feature) is Err(S) - Err(S|A) = 1, so we split on A first, the next splits don't matter which order you do them as they have the same error.
