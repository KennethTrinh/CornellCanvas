My understanding is that some of the features that really help the training set aren't as helpful on the test set (e.g., time). I was able to get a better understanding of the data by graphing some of the features in the test and train and seeing potential areas that the two don't overlap or do.

Also, because this dataset is so small, I would test using a smaller k, like 5 so that you don't overfit on the data (I know you said it doesn't really matter, but this should be helpful in general).
