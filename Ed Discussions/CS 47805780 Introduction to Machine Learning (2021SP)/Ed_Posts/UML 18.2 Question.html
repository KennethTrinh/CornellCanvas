Whow! I think you found a typo in the algorithm!

Here is what I think this line should read. Instead of checking "if all examples in S have the same label", we should be checking "if all examples in S have the same value for feature $x_j$" (i.e. the feature with the largest Gain). Here is the reason: if this feature does not further partition the training data, then there is no point in growing the tree any further. In particular, if we kept going we would have to hallucinate a label for the empty branch of the tree in the next level of the recursion, and for the other branch of the tree the other attributes we could be further splitting on would have even lower Gain for exactly the same S as in the current iteration.
