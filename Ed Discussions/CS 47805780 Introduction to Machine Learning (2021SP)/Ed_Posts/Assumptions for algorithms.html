I would say that an example of the above is that for a vanilla linear classifier, that we make the assumption that our data must be linearly separable, otherwise we can't draw a consistent hyperplane. Off the top of my head, I don't know if there are any other assumptions we make about data other than maybe for the same set of features we assume so far in the course the output label is deterministically determined.
