You definitely need both the conditional and the marginal probabilities, otherwise, you wouldn't be able to compute the argmax. $\vec{w}$ can be created with just the conditional probabilities, and $b$ with the marginal probabilities, so neither add new parameters to the multinomial bayes. In summary, the parameters are: $P(W = w_{i} | Y = y)$ for each $i$ and $y$ $P(Y = y)$ for each $y$ (Note that some of these parameters can be obtained from others using the laws of probability).
