Not sure if this answers your question, but taken directly from the Perceptron algorithm (lecture notes 7b), a point is misclassified if

y_i\left(w^{\left(t\right)}\cdot x_i\right)\le0

In general, when we want to classify a new data point, we label it +1 only if LHS strictly >0 (see lecture slides 6b)
