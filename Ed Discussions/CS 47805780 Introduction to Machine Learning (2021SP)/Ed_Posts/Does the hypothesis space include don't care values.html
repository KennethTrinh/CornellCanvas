If I only had one attribute, red or not red, and the label is good or not good, is the hypothesis space like this:

  • red -> good

  • red -> not good

    • Or is it like this:

  • red -> good

  • red -> not good

  • not red -> good

  • not red -> not good

    • Or is it like this:

  • red -> good

  • red -> not good

  • not red -> good

  • not red -> not good

  • () -> good

  • () -> not good

    $\mathcal{H} = \{ \{\{red \rightarrow good\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow ngood\}\} \}$

    It has four elements and each element maps both $red$ and $nred$ to some label. Every element of the hypothesis space maps each element of the instance space to some label.
