$\mathcal{H} = \{ \{\{red \rightarrow good\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow good\}\}, \{\{red \rightarrow ngood\}\{nred \rightarrow ngood\}\} \}$ It has four elements and each element maps both $red$ and $nred$ to some label. Every element of the hypothesis space maps each element of the instance space to some label.
