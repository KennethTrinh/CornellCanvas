Do you mean "positive semi-definite"? But, it is. By definition, if a kernel function k can be written $$k(x,z) = \phi(x)^T\phi(z)$$ then k is a kernel.

Consider also that the kernel matrix K made with the entries $K_{ij} = k(x_i, x_j)$ is a positive semidefinite matrix. If you had a vector $p$ such that $p_i = f(x_i)$ then $K = pp^T$, and from there you can see that K must satisfy the definition of positive semidefinite: $x^TKx \ge 0\ \forall x$ since $x^TKx = x^Tpp^Tx = (p^Tx)^Tp^Tx \ge 0$.
