1. Consistent starts considering the whole hypothesis space and constrains it after each example to what was consistent with the x given at each time step. To make |H|-1 mistakes you must choose every wrong hypothesis before landing on the right one, which means that there is exactly one consistent hypothesis, and only one hypothesis is marked as inconsistent at each time step. Note you can enumerate these for a small |H| since it was not given. 2. Similarly for this, you can show the way the space gets halved and how that affects the mistake bound. You have the right idea here.
