I am unclear about the precise definition of the instance space and its size from the lecture. If there are three traits in X all of which could take on 3 values each, is the instance space defined as the set of those three traits or the set of all 3 x 3 = 9 possible trait values? And so would the size of the instance space be 3 or 9?

The size of the instance space is the number of possible inputs you can have. Let's say you have a set of features $F$. If you have a total of $|F|$ discrete features $A_1, A_2, \dots, A_F$ and each feature $A_i$ has $|A_i|$ possible feature values, then the size of your instance space is $$ |X| = \prod_{i=1}^{|F|} |A_i|$$ This number captures all possible inputs in your input space $X$. For example, in section this week we had $F = \{\text{Taste}, \text{Smell}, \text{Crust}, \text{Appearance}\}$. Since Taste could be good/bad, Smell could be spiced/fruity/woody, crust could be firm/soft and Appearance could be good/bad, we have $|Taste| = 2$, $|Smell| = 3$, $|Crust| = 2$ and $|Appearance| = 2$. Therefore, $$ |X| = |Taste| \times |Smell| \times |Crust| \times |Appearance| = 2 \times 3 \times 2 \times 2 = 24$$. And so to answer your example, it would be $3 \times 3 \times 3 = 27$, not 3 or 9.
