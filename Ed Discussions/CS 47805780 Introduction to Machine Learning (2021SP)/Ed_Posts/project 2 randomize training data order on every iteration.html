The instructions say to randomize "on each iteration," so I think we have to randomized after each update.

I'm not sure I understand the second question. I think the algorithm should always stop after iterating max_iter times, regardless of whether it classifies the data correctly or not.

For the last question: Once the perceptron correctly classifies all the data, there's no point in going through the data again because you won't misclassify any point, and, as a result, you won't make any updates. Therefore, it doesn't matter whether it counts as an iteration or not because, if that happens, then you're done training.
