Think about the dimensions, xTr is 100*2, suppose you have W 2 * 20, the result is 100 * 20, which is what we want for the next layer.

If that does not make sense, try first without the "100", but think of an individual sample. x has 2 dimensions, we want to have 20 "results" by weight x1 and x2 differently and sum them. Naturally, we want 20 groups of such weights, each with 2 entries (w1, w2). Given the row representation of vectors, this means we want a 2*20 matrix for all these weights.
