Q1: Not 100% sure what you mean. You want to make sure the functional margin is 1. The margin is usually calculated as the distance between a line (defined by $\mathbf w, b$) and a few specific points. Not all points on the plane will have the same distance to the hyperplane!

Q2: If I have an equation $\mathbf w \mathbf x = b$, then any point $\mathbf x$ on the line it represents will satisfy this equation. This is the definition of a line. If I scale $\mathbf 2$, say by some constant $c$, will the same set of points satisfy the equation?

i.e. $\mathbf w = (1,1), b = 1, c = 2$

Line 1: $(1,1) \cdot \mathbf x = 1$

Line 2: $2 (1,1)\cdot \mathbf x = 1$

Will every $\mathbf x$ that satisfies the first equation also satisfy the second, and vice versa? If so, then they represent the same line, and you do not need need to scale $b$.

Author: Jane Du (staff)

Vote Count: 1

The functional margin is defined as the smallest distance from a training point to the hyperplane. In other words, the distance from the plane to support vectors. There is no such thing as a "functional margin for all points". No, w and b should not be functions of x. (please follow up if it's not clear!)
