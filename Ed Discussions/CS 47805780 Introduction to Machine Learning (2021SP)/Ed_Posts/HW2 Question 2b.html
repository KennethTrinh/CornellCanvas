The standard form is $w_{1}x_{1} + w_{2}x_{2} + b = 0$, so please use this form. Apologies if this was not clear beforehand! Keep in mind that if you get a solution of the form $w_{1}x_{1} + w_{2}x_{2} = d$ where $d$ is a constant, then you should submit $b= -d$ as your result.
