I'm not sure how to define a weight vector when input dimensionality can vary. How can there be a "universal" weight vector/hyperplane for all inputs if inputs can have a variety of dimensions?

----------- REPLIES -----------

Author: Bonnie Akhavan (staff)

Vote Count: 1

All inputs will have the same dimension $d$, we just haven't given you an explicit integer for $d$. For constructing your weight vector $w$, it's fine to write something like $w = [ w_{1}, w_{2}, ...., w_{d}]$ (replacing those values with numbers) since you don't know what $d$ is.
