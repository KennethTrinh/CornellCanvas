All of H are circles, so to shatter a set of 1 point looks like this: To shatter 2 points with H, I need to look at 2^2 labellings, and it looks like this: And shattering 3 points is also doable. I haven't drawn out all 8 cases, but you can see some of them: Recall that for a VC dimension to be d: 1. There exists at least 1 set of size d that is shatterable -- i.e. for all 2^d labellings of this set, I can represent them using hypotheses from H. 2. For all sets of size d+1 and above, there is no way to shatter them. i.e. there exists at least one labelling of these d+1 or more points that can't be captured from H. Typically point 2. above is harder to prove. Note: if there exists a set of size d or below that is not shatterable, such as 3 colinear lines with the circle, it does not preclude point 1. In other words, the VC dimension is at least 3 even if there is only 1 very specific very ideal set of 3 points shatterable by H.
