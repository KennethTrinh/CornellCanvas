Author: Jane Du (staff)

Vote Count: 0

Just double checking, but: P(Y=-1 | X=011) + P(Y=+1 | X = 011) should equal 1, yes. The expression P(Y = -1, X=x) + P(Y = + 1, X = x) should equal P(X=x).

And the equation from the link I sent should translate to P(Y=-1 | X=011) = P(Y=-1, X=011)/P(X=011). Again, this is just the definition of conditional probability. Note that you have to add the two joint probabilities (the expression with the comma) to get P(X=x).
