Hi everyone, Project 3 is now released on Vocareum. This project is due April 2nd, 11:59PM, and is an individual project. There is no late submission allowed. If you have any questions, feel free to reply to this post or create a new question on Ed Discussion. Below is an FAQ that we will update as questions pop up. Please start this project early. Some of the formulas can be confusing, so it would likely serve you well to get a head start. Clarifications/FAQ The equation "yi (∑αi yi k(xi,x) + b)=1" is somewhat confusing, since there are in fact two different indices. A more clear equation would be "yi (∑αj yj k(xi,x) + b)=1". Some students have been getting the error "vocSaveDataPublic not found". This is a typo on our part, but should not affect your submission and you can ignore it. If you see this error and are not getting full points, it is likely due to some other issue like timeout. Sorry for the confusion.
