S U (x,y)) = S with the new example (x, y) added to it VS(H,S U (x,y)) = the version space with S with the new example (x, y) added to it VS(H,S U (x,y)) C VS(H, S) = the version space with S with the new example (x, y) added to it is a subset of the old version space, before (x, y) was added If you add the right (x, y) to S, you can restrict the version space a lot. For example, if you only had positive (x, y)s before, the version space includes any hypothesis that forms a circle that surrounds all those points, no matter how big that circle is. But if you add a negative (x, y) farther away, now the version space can only include circles that surround all the positive points but does not include the negative one. That limits it a lot from what it was before.
