When l = d-1 (you're updating the weights on the last layer), you get the formula: W_{d-1}^{\left(t+1\right)}\leftarrow W_{d-1}^{\left(t\right)}-\eta G_{d-1} and G_{d-1}=v_{d-1}D_d where D_d=\sigma'\left(s_d\right)\frac{\partial L}{\partial v_{d,1}} So, you can see that when updating the last layer of weights, we are including the loss function w.r.t. the output of the last layer. If you look at all the other layers, you'll see they also include the term D_d, so in fact, in all the layers update rules you have one big chain rule with the loss function w.r.t. each layer's weights.
