Author: Artem Streltsov (staff)

Vote Count: 0

This is a discrete feature space and we assume we know all possible values each feature can take. In such a setting the instance space is simply generated via a cartesian product of all unique attributes. The hypothesis space is then the set of all possible maps from this instance space that contains all combinations of attributes to {yes,no}. All features are used, but the instance space already has subsets that a hypothesis can map to the same constant as if a particular feature does not matter.
