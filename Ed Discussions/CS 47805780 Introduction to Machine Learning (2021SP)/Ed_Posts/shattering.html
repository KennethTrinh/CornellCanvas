Author: Jane Du (staff)

This is a great question.

In short, yes, you need to fix them. But, you can fix them anywhere you like.

e.g. hypothesis space of circles, where the inside is positive and outside is negative, cannot shatter 3 points in a line like + - +. However, if you moved the middle point up in space a little, suddenly you can get all possible combinations using circles. So, circles can shatter a set of 3 points.

Then to show that the VC dimension is 3, you'd have to prove that no 4 points is shatterable by circles no matter how you initially place (and then fix) them.
