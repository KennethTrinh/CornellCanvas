In general, what is the difference between a test set and a validation set?

----------- REPLIES -----------

Author: Victor Butoi (staff)

Vote Count: 0

Test set is a portion of your data that is set aside until the very end of your training cycle, that you never see until you need to know how well your model generalizes to the real distribution. Validation set is something that you do not train on, ie doesn't update model weights or parameters, but you evaluate trained models on so that you can tune hyperparameters- because the implicit belief is that there exists a best set of hyperparameters for you given problem, but that if you were to find these using your test set, then the hyperparameters would overfit that test data and it wouldn't represent accuractly how well the model generalizes.
