    Author: Brandon Guo (staff)

    Vote Count: 1

    For clarification, you are not allowed to "look up" the exact hospitalization numbers of the date you are trying to predict. For example, you are not allowed to use the t-values from a date one month from now to predict the cases today, but you are allowed to use the data one month in the past to predict cases today. This actually doesn't apply to train_baseline.csv because there are no t-values in that file.

    This means you can use any of the features provided in a example to predict the value of that example.
