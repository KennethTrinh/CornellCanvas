Author: Carlos Alvarez (student)

Vote Count: 0

Since the dimension of the input is of dimension D, and the dimension of the first layer of the network is of dimension D, then there are D^2 weights connecting the input to the first layer.

------------------------------------

Author: Nina Tang (staff)

Vote Count: 0

The network has D input nodes fully connected to D nodes on the hidden layer, then connected to C output nodes (3 layers including the input and output nodes). So the D^2 is for the weights between the input D nodes and the hidden layer D nodes.
