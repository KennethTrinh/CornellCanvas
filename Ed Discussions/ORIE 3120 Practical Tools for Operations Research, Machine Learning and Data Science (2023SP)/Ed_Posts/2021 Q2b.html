Why do we need to reset_index() in this problem? SELECT OrderID, SUM(Quantity*UnitPrice*(1-Discount)) FROM OrderDetail GROUP BY OrderID orderdetail['revenue'] = orderdetail['Quantity']*orderdetail['UnitPrice']*(1-orderdetail['Discount']) orderdetail.groupby('OrderId')['revenue'].sum().reset_index()

----------- REPLIES -----------

Author: Varun Gande (staff)

Vote Count: 0

If we don't reset index, OrderId will be an index column and not an actual column of the dataframe. Meaning, you could not reference the OrderId column by calling df['OrderId'].
