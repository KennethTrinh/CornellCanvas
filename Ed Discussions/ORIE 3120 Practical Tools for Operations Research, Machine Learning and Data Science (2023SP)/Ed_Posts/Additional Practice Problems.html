1. They seem interchangeable but we typically use taxi['trip_distance']

for setting, values, you need to use df['column'] = series .

once this is done however, you can refer to that column in the future with df.column , assuming it's a valid python name. (so df.column works, but df.6column would still have to be accessed with df['6column'] )

see this link for more info https://stackoverflow.com/questions/23546555/in-pandas-whats-the-difference-between-dfcolumn-and-df-column

2. & and | are used in pandas. AND and OR are used in python (will not work in pandas). If comparing things in pandas dataframes, use & and |.

see this link https://towardsdatascience.com/bitwise-operators-and-chaining-comparisons-in-pandas-d3a559487525

3. the ON= needs to be the column that is the same name in both dataframes A and B.
