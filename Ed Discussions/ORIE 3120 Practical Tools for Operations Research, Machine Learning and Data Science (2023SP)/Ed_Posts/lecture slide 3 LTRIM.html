Author: Julia Vanputte (admin)

LTRIM will remove the specified characters from the LEFT side of the string. Since the left side of the string is 'O...' and does not contain any characters in '3120', these characters will not be removed. Ltrim will trim all characters in the specified string to trim until another character on the left is seen.

Author: Varun Gande (staff)

Vote Count: 1

You're going to trim characters on the left side as long as they are in the second argument. So if the second argument was 'RO', we would trim an O (since it's an O or R), then we trim the R (since it's an O or R), but we don't trim the I (since it's neither an O nor an R). Therefore, the trimmed string is IE 3120.

If the second argument of LTRIM was '3120', we keep trimming characters as long as they are a 0, 1, 2, or 3. The first character of 'ORIE 3120' (O) is neither a 0, 1, 2, nor a 3, so nothing gets trimmed in this case.
