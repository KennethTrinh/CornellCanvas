On some of the practice prelims, it mentions that there is an equivalent WHERE and JOIN statement. Does WHERE do the same thing as one of the JOINS, or is it more complicated than that?

Author: Samantha Klein (staff)

Vote Count: 0

WHERE is typically used to filter data. In some cases, WHERE can also be used to join data. For example you could have a query: SELECT * FROM x, y WHERE x.name = y.name Here, you use a WHERE clause to define the join. This is an implicit join where you can join on a particular feature.
