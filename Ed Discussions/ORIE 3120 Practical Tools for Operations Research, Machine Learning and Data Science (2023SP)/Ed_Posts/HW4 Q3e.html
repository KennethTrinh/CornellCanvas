Author: Julia Vanputte (admin)

Vote Count: 0

Coalesce returns the first non-null value in a list, so I would not suggest that function in this case. For dealing with the NULLs, think carefully about how to join your data. Additionally, there are quite helpful functions regarding NULLs provided by SQL: ISNULL() and IFNULL().
