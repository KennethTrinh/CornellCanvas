You might want to think about whether your dataset has NULL values or not. For example, if your dataset has three records with NULL values, SQL would count those as three seperate records, instead of consolidating those three records into one NULL record. So if your dataset does have NULL values, you would need to consider using the DISTINCT keyword, as the HW states.
