In this case, the way you should read this is "the correlation between an element and the one k elements further is the y value". It makes sense that the value is 1 at 0, since an element and the one 0 elements after it (aka itself) will always be the same, and the correlation will be 1. If you have a significant correlation elsewhere (generally, anything going far beyond the shaded area), then there is likely correlation between those elements (e.g. some element and the one 10 elements after it, if you have a spike in either direction at 10). What Qian and I meant with that line is to think about "how is each value generated? What are you using, is there pairwise independence, are residuals independent of the position on the line, etc". Hope this makes sense!
