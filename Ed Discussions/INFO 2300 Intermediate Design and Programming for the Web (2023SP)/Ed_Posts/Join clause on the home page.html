The answer depends on your design.

If you display the tags for each entry on the view all page, then you will need a separate query for each entry with a JOIN to get the tags. If you're not displaying the tags for each entry on the view all page, then no JOIN is needed.

You are required to list the tags on the details page. Retrieving the tags will require a separate query with a JOIN clause.
