This is typically handled two ways:

1. Server-side image scaling and cropping using a background job. This is an advanced topic that we won't cover in this class.

2. CSS to resize and crop/frame images. Please check out the reference documentation for this. There are many ways to do this. A quick web search comes up with this, which has many viable solutions: https://stackoverflow.com/questions/15167545/how-to-crop-a-rectangular-image-into-a-square-with-css
