Hey! I was the TA that checked your installation the other day. Two other things you can try: Copy these four lines into the end of your ~/.zshrc file and restart terminal. # Add Visual Studio Code (code) export PATH="$PATH:/Applications/Visual Studio Code.app/Contents/Resources/app/bin" # Add Docker Desktop for Mac (docker) export PATH="$PATH:/Applications/Docker.app/Contents/Resources/bin/" Try the following command: /Applications/Docker.app/Contents/Resources/bin/docker --version If this works, you just need to run: ln -s /Applications/Docker.app/Contents/Resources/bin/docker /usr/local/bin/docker Then, restart terminal and try to type a docker command again, like docker --version . Hopefully this works. If you're still stuck, then please come to office hours!
