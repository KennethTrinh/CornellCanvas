You may, for example, have your tag filters showing up on the page as <a> tags or radio buttons within a form. What we don't want to see is a hard-coded <a> for each tag (or, in the case of the form, a hard-coded radio button for each tag). Instead, you should retrieve all tags, then loop through what you retrieved and echo the content you want (such as id or tag name) accordingly.

Does that make sense?
