<a> elements are easier for this. It's just like what we did for sorting the reviews page in class. However, if you want to use a drop-down, see the update grade form we used in class today. Observe, there is a <select> element with a list of grades. Remember here that a form with a GET requests puts all the parameters as query strings (POST requests do not). Go ahead and try changing the change grade form to a get request and take a look at the URL. Notice after submitting the form, the grade is in the URL! Note that the change grade form uses an associate array for the grades. You won't want to do that because all your tags are in the database. Instead, you want to query the tags table and put each tag record as the <option> s in your select box.
