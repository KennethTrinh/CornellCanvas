Is it possible to make a form sticky with a <textarea>? Looking at the documentation, there doesn't seem to be a 'value' attribute for <textarea> like <input> does, so I was wondering how to display the already submitted information in this <textarea>.

The value attribute sets the initial value for <input> which doesn't have a closing tag. With <textarea>, you have both opening and closing tags, so you can set the initial value just by writing it in between the two tags. I hope this helped!
