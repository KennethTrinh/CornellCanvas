    Good questions!

  • There can be an infinite amount of tags per item.

  • For the insert form, you should not use radio buttons since it only permits one tag.

  • On the view all page, you need to be able to filter by one tag.

  • All tag and entry data should be retrieved from the database; no tag or entry data should be hard-coded.

  • On the insert form you may use checkboxes for the tags, if you like. There are other solutions as well. You should use the tag data from the tag's data to label these boxes. You should not use key value coding.

  • You will need to associate each check box with the tag id using the tag records that you query.

    • I hope this helps! Please let us know if you have any other questions
