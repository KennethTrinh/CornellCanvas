In general, to keep things simple, if you want to put all your code changes on GitHub (the website with your repositories) via a command line (e.g. terminal), you can follow these steps: First, navigate to the project directory, e.g. cd ~/info2300/lab1 . Then: git add <file name(s)> # For example: git add . <- this will add all files in the current directory to be committed git commit -m "Title of commit message" # For example: git commit -m "Changes to Lab 1 website frontend" git push # Usually this will work without issues, however if you see some message about "failed to push upstream" then try following the suggested fix in the message and try again.
