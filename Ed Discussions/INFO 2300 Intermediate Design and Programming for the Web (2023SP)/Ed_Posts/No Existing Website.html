To meet the pre-req for this class, you should have created a static website at some point. If you don't have a static website, then it's likely that you haven't submitted a waiver for the pre-req. Please note that students without the pre-req will be dropped from this class by the end of the add period if they don't have a 1300 waiver.

1300 students start with a blank repository. So there is no example "blank INFO 1300 Project 1" to provide you with. You should see the best practices section in the 2300 Project 1 requirements for most of the conventions.

Your website should be created using the user-centered design process. This means conducting interviews, etc. You should have done this to meet the pre-req of 2300.
