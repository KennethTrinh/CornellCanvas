Generally, localhost:8001 is used when port 8000 is already in use (e.g. another php server). Check that you are on the right port for the code you are looking at. An easy way to close php servers run through vscode is to restart vscode.

The favicon should only show up if it is in your code- if it is appearing erroneously, try restarting the php server and clearing your cache.
