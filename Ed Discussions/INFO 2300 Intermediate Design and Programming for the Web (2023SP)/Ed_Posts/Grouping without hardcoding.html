I suggest added a key-valuing code field to categorize tags: 
tags (
  id,
  name,
  category
)

const TAG_CATEGORY = array (
  1 => 'Rapids',
  2 => 'Permits',
  3 => 'States'
);
Now, when you render the tag records, you can use the key value coding to present the different tag categories to users.
