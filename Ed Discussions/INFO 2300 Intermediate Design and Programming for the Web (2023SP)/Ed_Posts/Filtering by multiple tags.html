No grouping of tags is required. In fact, I recommend against it. It is unnecessary for full credit, and it adds additional time and testing for a time intensive project. You can just have the tags (even if they are different "groups") in any order, for example: #red #tomato #yellow #cucumber #purple #melon ^ these tags aren't grouped and this 100% meets the project requirements.
