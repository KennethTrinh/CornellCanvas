It means not to set a variable to the empty string('') or 'hidden' and echo that out as a class. But rather use conditional statements in PHP to conditionally output some HTML. This was covered in Class 3/8.

For example: 

<?php if (TODO_some_expression) { ?>

  <p>this HTML will only output if TODO_some_expression evaluates to true.</p>
  
<?php } ?>
