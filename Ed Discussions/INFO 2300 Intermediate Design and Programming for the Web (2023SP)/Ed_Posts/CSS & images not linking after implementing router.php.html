I'm sorry that you've gone to so many office hours and weren't able to figure out what's wrong with your code. I know this has been a frustrating experience for you.

I've reviewed your code, and noticed that your code is outputting this on top of every single file (HTML, CSS, Images, etc.):

<!-- Source for all code in this `router.php` file: Kyle J. Harms (copied with permission for INFO 2300 project 1)-->

This HTML comment turns every single file into an HTML file. View the source for your CSS in your web browser. You'll see this HTML comment. You're also putting this on every image.

Please note that the class notes for routing explicitly state that no citation is necessary for the router.
