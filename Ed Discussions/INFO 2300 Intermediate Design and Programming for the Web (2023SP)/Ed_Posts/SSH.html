Hi! For SSH key , you only need to configure SSH once for all of your classes. Once you have configured your computer to authenticate with Cornell's GitHub it will work for all classes, clubs, etc. that use Cornell's GitHub. For SSH agent , you need to configure it to make sure Git can work inside a container.

In the SSH Key configuration section, see the red box asking you to jump to step 11. If you're able to run the command ssh -T git@github.coecis.cornell.edu successfully, then your keys are configured, and you should not re-configure them.
