Hello, You can either convert the value to a string before sending it to the serial port or let the print function do it for you. Either solution will work. Once you tested your code manually, you should: close the serial monitor; Start the processing app; Observe the result on the screen; Note that if you see an error message on the processing side, it is probably because you have to adjust the serial port.
