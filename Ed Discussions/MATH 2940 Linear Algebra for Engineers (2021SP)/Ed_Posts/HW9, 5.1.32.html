The good thing about this sort of problem is that you can visualize it. Instead of writing down any matrices, try imagining rotating the unit sphere in $\mathbb{R}^3$ about the $z-$axis. Are there any points that don't move? Remember that for a transformation $T:V\to V$, $v\in V$ is an eigenvector of eigenvalue $1$ if (and only if) $v\ne 0$ and $Tv=v$.
