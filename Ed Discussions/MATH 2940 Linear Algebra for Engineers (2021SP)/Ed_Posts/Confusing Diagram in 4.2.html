Null spaces are a vector space, meaning they can be defined as a Span of some set of vectors. So if that set of vectors is linearly independent and contains one vector, then all vectors in the Null space must be collinear. If it contains two, then they're all coplanar... and so on.

If the Null space is just the zero vector, then it's just one point.
