Spanning all of $\mathbb{P}^2$ occurs if and only if you can span all possible coordinate vectors (which live in $\mathbb{R}^3$ since $\mathbb{P}^2$ is 3 dimensional). Hence if a set of coordinate vectors $[v_1]_\mathcal{B},\ldots,[v_p]_\mathcal{B}$ span $\mathbb{R}^n$, then $v_1, \ldots, v_p$ span $V$.
