This Theorem tells us that in order for a m x n matrix to span Rm it must have exactly m pivot columns . The reverse (if it has m pivots it must span Rm) is also a true statement .

In case 1 he proves that if there are exactly m pivot columns the matrix must span Rm.

My issues is in case 2. He shows that if you have the particular case where f is has a non zero bottom entry then the matrix must not have a solution hence must not span Rm.

But that is a particular case, and if you choose an f with a zero bottom entry then it can span Rm . So then I see a contradiction because the theorem says you cannot span unless you have only m pivots, but it is possible to span if you have less than m pivots and a particular f with a zero bottom row.

Note: Ok I feel really silly now but I realize my mistake. The definition of span means it covers all vectors, not that it is consistent for any particular vector. By showing there is a case where it is inconsistent that shows it does not span all vectors.
