I wrote a proof which uses the same reasoning as OP, but refers specifically to a theorem. Would the following receive full credit, or do we need to identify weights for w exactly as the solution writes it? We know (by a theorem in the textbook) that if a list of vectors v1...vn is linearly dependent and v1 is not zero, then some vj for j>1 is a linear combination of the preceding vectors v1...v(j-1). Since {u,v} is linearly independent, u cannot be 0. Also, v is not a linear combination of u. So, because the set {u,v,w} is linearly dependent, u=/=0, and v is not a linear combination of u, then w must be a linear combination of {u,v}.
