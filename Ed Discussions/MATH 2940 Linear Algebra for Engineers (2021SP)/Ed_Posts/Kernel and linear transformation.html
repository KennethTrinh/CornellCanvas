You can check it directly. If $T:V\to W$ is a linear transformation between a pair of vector spaces, we show $\ker(T)$ is a subspace of $V$. If $v,v'\in \ker(T)$ then $T(v+v')=Tv+Tv'=0$ and if $\lambda \in \Bbb{R}$ is given $T(\lambda v)=\lambda Tv=\lambda \cdot 0=0$. So, $\ker(T)$ is a subspace.
