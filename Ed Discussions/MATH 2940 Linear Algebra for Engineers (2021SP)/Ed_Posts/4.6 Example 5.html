Any solution $x$ solves all the equations of the system $Ax=0$, regardless the number of equations. The solution set imposes restrictions on the variables and not on the number of equations. The problem indicates that the whole solution set can be described through linear combinations of two linear independent nontrivial solutions, so $dim(nul(A)) = 2$.
