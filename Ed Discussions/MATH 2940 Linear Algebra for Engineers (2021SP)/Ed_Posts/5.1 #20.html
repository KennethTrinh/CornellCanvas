Author: Esther Wang (student)

The matrix is entirely 5's. Imagine a column vector with entries (a,b,c) where a+b+c = 0. Then the matrix vector product will be (5a + 5b + 5c, 5a + 5b + 5c, 5a + 5b + 5c) = (5(a+b+c), 5(a+b+c), 5(a+b+c)) and since a+b+c = 0, the resulting vector will be (0,0,0) which is 0 * the original vector.

Author: Esther Wang (student)

Vote Count: 1

Eigenvectors are defined such that Ax = λx with x (the eigenvector) not equal to 0. Given that the matrix is clearly not invertible, we know one such λ = 0. That is, Ax = 0x or Ax = 0. That is, the matrix vector product between A and our eigenvector has entries all 0. By definition of matrix vector product, each entry will be 5*(a+b+c). We've already established that this must be 0. So the entries in the eigenvector x (a b and c) sum to 0.
