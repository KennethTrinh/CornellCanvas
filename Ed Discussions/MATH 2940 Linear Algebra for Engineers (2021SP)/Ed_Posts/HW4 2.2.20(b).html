Yes, I think the converse is also true but not entirely sure. Let C=AB. We know that AB is invertible, so inverse of C exists. Then if you multiply inverse of C to the back of both sides, it becomes I=\ A\left(BC^{-1}\right) We can see that that A has some inverse (BC^-1) whose product makes it into I and hence, it's invertible. Similarly, if we muliply C to the front of both sides. This is... I=\left(C^{-1}A\right)B Once again, B has some inverse that makes the product into I, so it should be invertible. I found this proof somewhere on the internet a while ago because I was wondering about the same thing.
