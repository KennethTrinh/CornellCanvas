You should commit the general solutions of the dynamical system and differential equation to memory so that you can figure out behaviour from first principles. If $\vec{x}_{k+1} = A\vec{x}_k$ and $A$ has eigenpairs $(\lambda_1,\vec{v}_1),\ldots,(\lambda_n,\vec{v}_n)$ and $[\vec{x}_0]_\mathcal{B} = \begin{bmatrix}c_1 & \ldots & c_n\end{bmatrix}^T$, the general solution of the dynamical system is $$\vec{x}_k = c_1\lambda_1^k \vec{v}_1 + \ldots + c_n\lambda_n^k \vec{v}_n$$ Suppose $\lambda_1 = 1$ and all $c_i$'s are zero except $c_1$. Then $$\vec{x}_k = c_1\lambda_1^k \vec{v}_1 = c_1 1^k \vec{v}_1 = c_1\vec{v}_1 \hspace{1cm} \forall k$$ So any vector that touches the subspace $Span\{\vec{v}_1\}$ stays in place forever. The steady state vector for a stochastic matrix in Markov Chains is an example of this. Now if we have a real matrix $A$ with complex eigenvalue $\lambda_1 = re^{i\theta}$ where $r=1$ and complex eigenvector $\vec{v}=\vec{x}+i\vec{y}$, there is a corresponding conjugate eigenpair $\lambda_2 = re^{-i\theta}$ and $\vec{w}=\vec{x}-i\vec{y}$. Assume all $c_i$'s are zero except $c_1$ and $c_2$. Then $$\vec{x}_k = c_1\lambda_1^k \vec{v} + c_2\lambda_2^k \vec{w} = c_1 1^ke^{ik\theta} \vec{v} + c_2 1^ke^{-ik\theta}\vec{w}$$ Assuming your initial vector $\vec{x}_0$ has real entries, the coordinates $c_1=a_1-ib_1$ and $c_2=a_1+ib_1$ must be in conjugate pairs, and if you expand everything out with sines and cosines, you'll get that $\vec{x}_k$ orbits the origin in closed elliptical orbits in the plane $Span\{\vec{x},\vec{y}\}$. The origin is classified as a "center", and you can determine whether the orientation is clockwise or counterclockwise by computing $\vec{x}_1$ and seeing which direction you changed angle by $\theta$. For the differential equation, the above cases happen when the real part of the complex eigenvalue is zero instead of absolute value equalling one.
