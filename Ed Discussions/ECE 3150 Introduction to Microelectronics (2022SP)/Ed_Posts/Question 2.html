The collector sees both R_c and the input impedance of the next stage, so that is why the second stage common collector was needed, so the 8 ohm load looked a lot larger to the common emitter stage.

You might have noticed a circular relation between gm and R_c. Gain increases with R_c, so choosing a smaller I_c lets you make R_c larger and still not bring V-c below saturation. But lower current reduced gm directly, so reducing I_c to boost R_c turns out to be a complete wash. Any current I_c you choose will result in the same gain if you set Vc to be identical in each case.

But design is always a set of trade-offs. Choosing a higher Ic means a lower Rc, and when it get paralleled with the input impedance of the next stage, it is not loaded as much, so you might actually achieve a higher net gain by raising Ic in the first stage. I emphasize 'you might"....

There is, naturally, a trade-off if you elect to increase I_c. Higher I_c reduced r_pi, so the input impedance of the first stage is reduced, and so there is an increased loss of input signal in the 100 resistance of the microphone.

All design involves trading one parameter for another. Often you can find an optimum between all the trade-offs, but as you can see that the process is usually iterative, and there will be many reasonable solutions to the same problem.
