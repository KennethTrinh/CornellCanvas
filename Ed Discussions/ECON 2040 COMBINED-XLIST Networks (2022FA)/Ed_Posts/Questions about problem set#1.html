(1) When a node is taken out, so are all the edges that leave/enter the node.

(2) Yes, a single node with no edges still fits the definition of a connected component: "a subset of the nodes such that: (i) every node in the subset has a path to every other; and (ii) the subset is not part of some larger set with the property that every node can reach every other" (Chapter 2)

(3) If there are multiple nodes that fit the property, you are only required to mention one example.

(4) I believe a scene should only be "geometric arrangement of points and filled-in polygons" as defined in question 2. --If other class staff know for sure, please add to my answer.

(5) A "strong tie between each pair of people" means everyone on the floor has strong ties with each other, your second example. Brandon would have a strong tie with Marcus, Alissa, and Taylor (everyone on their floor). Similarly, Marcus would have a strong tie with Brandon (as mentioned before, because it goes both ways), Alissa, and Taylor too.
