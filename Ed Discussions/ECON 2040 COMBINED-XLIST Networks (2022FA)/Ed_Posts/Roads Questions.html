How would you determine if a road is strictly dominated or not?

----------- REPLIES -----------

Author: Caleb Chin (admin)

Vote Count: 0

There is no general process, but a road is strictly dominated if it takes strictly longer to take that path than any other alternative path that a user could take at a particular junction at any point in time.
