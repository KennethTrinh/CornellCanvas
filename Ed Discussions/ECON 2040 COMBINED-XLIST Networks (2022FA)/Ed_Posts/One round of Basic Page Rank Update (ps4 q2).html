In a graph, there is no sense of "up" or "down" - all nodes are equal, and should be treated as such. The Basic PageRank Update affects all nodes - you should calculate how the PageRank will flow along the edges, and update each node accordingly. Not sure if this is the confusion, but each node changes simultaneously (i.e. the PageRank value for some node in Step 2 only depends on the PageRank values of other nodes from Step 1)
