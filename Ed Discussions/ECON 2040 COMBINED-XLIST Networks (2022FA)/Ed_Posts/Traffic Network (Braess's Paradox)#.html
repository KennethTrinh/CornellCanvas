Author: Caleb Chin (admin)

In this example, the creation of a 0 minute C-D road essentially splits problem into two decisions for drivers. 1. what is the best way to get to from A to C/D? 2. what is the best way to get from C/D to B? Note that this is possible because drivers can switch between C and D without time cost. 1. If all the cars in this problem (500) went on the road A-C, the time it would take is 50 minutes. If we compare it to the constant road A-D (55 min), we can see that the A-C road is always preferable. 2. Using similar logic, we can compare the time for all drivers taking D-B with the time it takes to get from C-B (55 min). Since drivers are self-interested, they will always go for the route that will be faster for them (D-B). This gives the dominant strategy of ACDB.

Author: Caleb Chin (admin)

Vote Count: 0

So, in this graph, we know that taking the A-C route means that we must spend 1 hour travelling from C-B as we cannot switch to D. We already know that drivers can get to B in 3 hours, so x/100 must be less than or equal to 2 hours. That means that some drivers will arrive at node D. Further, we know that every driver that arrives at D can already get to B in 1 hour (through C). Thus, y/100 must be less than or equal 1 hr. If this y path is less than an hour, than people start to take to the y route more, leading to an increase in time until it is equivalent to the time the C-B takes. We can solve for y and make a similar argument to solve for x. Unfortunately, there is no definitive answer to when we can split up the problem into two sections like the original question. If there is a bidirectional 0 length C-D road in a simple 4 node diamond-like structure, we are pretty certain we can split up the graph. In other cases, we might have to run some analysis like the one we did above.
