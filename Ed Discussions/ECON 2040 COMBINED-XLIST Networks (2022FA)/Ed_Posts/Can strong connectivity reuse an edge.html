Not quite sure what you mean by a "cycle" and SCC "using" an edge.

The definition of SCC is that there is a path between every set of nodes in the set (as well as being maximal). If you use an edge twice in a path, then there was a cycle in the path, and you could just use the path without that cycle

To clarify:

  • The path 5-7-8-6 alone doesn't constitute a path that defines an SCC, but the path 5-7-8-6-5 does, as all nodes have a path to all other nodes (as does the other circular path you mentioned)

  • The set {5,7,8,6} is therefore an SCC because of the circular path above. SCC's are sets of nodes, not paths
