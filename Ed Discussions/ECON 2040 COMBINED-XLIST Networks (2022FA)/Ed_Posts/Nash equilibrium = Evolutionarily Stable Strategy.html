Nope, that logic doesn't quite hold: if A implies B, then (not A) doesn't necessarily imply (not B). However, a valid assertion would be (not B) implies (not A) (in this case, that would be saying that if A isn't a Nash Equilibria, then it can't be an ESS). Additionally, the counterexample above would also disprove this (the one where a Nash Equilibria is not an ESS)
