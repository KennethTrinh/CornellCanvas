Outside options: Node A = 0 Node B = 12-9=3 (value of edge-what the other node would take, in this case C or D) Surplus = value of edge - (outside option for Node A + outside option for Node B) = 12 - (0+3) = 9 Node A gets = outside option + 1/2(surplus) = 3 + 4.5 = 7.5 Node B gets = outside option + 1/2(surplus) = 0 + 4.5 = 4.5 Seeing as this isn't what we had before, it isn't a Nash Bargaining Outcome (One immediate error to note is that each edge is worth 12, but in your calculations, A and B splitting the edge would each get 9 and 6, which is more than the edge is worth)
