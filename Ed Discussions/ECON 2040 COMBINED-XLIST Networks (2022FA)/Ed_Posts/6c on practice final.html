If the relative values don't change, then the number of nodes with PageRank greater or less than don't change, as those are relative comparisons? It sounds like you're trying to compare absolute values from part c and part b - those are completely separate. We are concerned only with how the values of other nodes in part c compare to Node C's value in part c. The question asks you to compare the number of nodes with greater value between parts, not the actual values between the two parts, so that an acceptable answer might be of the form "There are X nodes in Part B with value greater than C in part b, which is greater than the Y nodes that have value greater than C in part c". Values are not compared across the parts, only the number of nodes
