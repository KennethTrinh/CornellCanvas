Edited: If you don't mind removing your Desktop from the OneDrive folder (in which case, it would stop being backed up), follow the instructions of the first method here .

If you would like to keep your Desktop folder backed up on your OneDrive folder, while still having that same folder appear in "its normal location," follow the following advice from Professor Lee (and found in Lab 3 activity 1): 

If you don't see "Desktop" in the ls output, your computer might have the Desktop folder
inside your OneDrive folder. (Check by entering ls OneDrive. If "Desktop" doesn't show 
up there, get help from a staff member.) Our recommended workaround is to create a 
"portal" from the Desktop in OneDrive to your home directory (this should allow 
OneDrive to still back up your Desktop). While in your home directory, enter 
New-Item -Name Desktop -ItemType Junction -Value OneDrive\Desktop
Now when you do ls, Desktop should be there, and you can continue with this lab!
