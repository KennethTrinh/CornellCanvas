Author: Dylan Castillo (None)

Yeah, .capitalize() will change the first letter of a string to uppercase and change the rest of the letters to lowercase. If you want to capitalize more parts of the string, you will either have to split it or have it split beforehand and call capitalize in each piece.

What I mean is something like this:

# You want string "Hello World" x = "HelLo WorLd" print(x.capitalize()) print("We don't want this! World is not capitalized.") #Instead, do this: a = "HelLo " b = "WorLd" result = a.capitalize() + b.capitalize() print(result)
