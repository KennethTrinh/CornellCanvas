The question is to count those that have no children, not to count each of the people. WIth reference to the testing code we provided (be sure to check it out! On the exams archive page..) 
grand = Person("grand", "parent") 
parent1 = Person("parent", "1", [grand]) 
parent2 = Person("parent", "2", [grand]) 
kids = [] for i in range(2): 
    kids.append(Person("kid", str(i), [parent1])) 
for i in range(2,4): 
    kids.append(Person("kid", str(i), [parent2]))
