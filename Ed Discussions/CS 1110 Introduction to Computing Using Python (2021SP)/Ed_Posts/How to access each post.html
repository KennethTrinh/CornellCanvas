If you've set the main tree up via [main_tree, main_dict] = main_example_tree() or [main_tree, main_dict] = a4_test.main_example_tree() then you can access a Post by mentioning its Gorn number: main_dict["0.0.1"] will return that Post. 
>>> import a4_test
>>> [main_tree, main_dict] = a4_test.main_example_tree()
>>> p010 = main_dict["0.1.0"]
>>> p010.author
'B'
>>> print(p010)
Post 010 by B:
    Reply ids/authors (if any):
>>> print(type(p010))
<class 'post.Post'>
>>> print(main_dict["0.0.0"])
Post 000 by C:
    Reply ids/authors (if any): 0000 by 'B'
