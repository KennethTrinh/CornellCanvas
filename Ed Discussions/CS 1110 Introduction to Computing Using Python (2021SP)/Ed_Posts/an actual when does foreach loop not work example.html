(extracted from today's office hours transcript) if you want to work directly with positions, use "for .i.. range (...)", because that directly gives you index numbers; if you just want to work with the values in a list, just iterate through the values, which are give be "for x in whatever_list" # this doesn't work (the way you expect) print("the attempt to use a for-each loop") thelist = ['hi', 'hi', 'hi'] list2 = ['nothing', 'nothing', 'nothing'] for s in thelist: i = thelist.index(s) print('i is: ' + str(i)) list2[i] = 'done' print(list2) print("finished attempt to use a for-each loop\n\n") #this works print("using range") thelist = ['hi', 'hi', 'hi'] list2 = ['nothing', 'nothing', 'nothing'] for i in range(len(thelist)): print("i is: " + str(i)) list2[i] = 'done' print(list2)
