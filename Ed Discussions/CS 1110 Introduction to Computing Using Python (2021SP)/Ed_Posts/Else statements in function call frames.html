Hi! The instructions say that "else" statements should be treated as executable (and [sometimes] executed) executed statements, just like any other line that might or might not be executed. See the archived 2018 Spring A2 assignment and solution for a worked example with an if-else statement. For your specific question, I can't tell what the indentation of line 3 is. Assuming it is indented, then the program/instruction counter would be 1, 4, 5, crossed out as appropriate. If line 3 isn't indented, then the counter would be 1, 3, crossed out as appropriate.)
