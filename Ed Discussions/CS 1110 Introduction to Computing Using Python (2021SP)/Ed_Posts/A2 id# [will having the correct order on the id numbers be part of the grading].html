But more importantly from an understanding-python point of view (as opposed to the grading policy): Python does not create all the global variables first, so you shouldn't do that either, tempting as that may be. 1. It's quite important to understand the flow of execution and creation of objects. 2. If some function call crashes, the "downstream" global variables wouldn't be created, but your diagram would have these "ghost variables" which can be read to imply that they had been created.
