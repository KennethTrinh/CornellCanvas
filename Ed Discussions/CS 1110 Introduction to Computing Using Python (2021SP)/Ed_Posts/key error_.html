Here’s an example: dictionary = {“you are a cool person”: “I know”} When you type dictionary[“this is a good question”], it would return in a key error, because “this is a good question” is not in dictionary. So, you would do the following: dictionary{“this is a good question”} = “indeed” Now, dictionary = {“you are a cool person”: “I know”, “this is a good question”: “indeed”} Hope this helps!!!’ Also since values for cdict are only college objects, you need to need to create college objects for colleges not already in cdict.
