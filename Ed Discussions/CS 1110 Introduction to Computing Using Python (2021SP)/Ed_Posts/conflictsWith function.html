conflictsWith is comparing the same course to itself (CS 1110), so both courses have identical times, so it should be finding a conflict. Make sure that your conflictsWith logic can handle this case, in addition to other cases. Thinking about the cases, the different ways a "conflict" can occur, is part of the exercise.

You can debug with guidance from https://edstem.org/us/courses/3933/discussion/434623 and https://edstem.org/us/courses/3933/discussion/430987?answer=1009083
