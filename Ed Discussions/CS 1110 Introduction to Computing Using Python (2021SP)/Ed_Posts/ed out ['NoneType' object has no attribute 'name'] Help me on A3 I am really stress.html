It says "'NoneType' object has no attribute 'name'". If you look at line 122 of helper_tests, and the lines just above it, you can see which test case is failing. In this case, the test is calling college_named and assigning that to result . Python then checks result.name , which throws that error, so result must be of type NoneType. This means that your function college_named is returning None instead of a college (even when the college exists), at least some of the time.

Make sure your function is always returning a college (if the college exists); if double-checking your logic doesn't help, then maybe try using debugging print statements to make sure that your variables all have the values you'd expect, or to check which lines are being run.
