So if the parameter parents is a list, we are creating an empty list and setting self.parents to that. Then we call self.add_parents(parents) to add the parameter list to our list of parents. Then for every parent that is in the list of parents, we need to add our self (our instance of class Person) to their list of children. That is what p.add_children([self]) is doing. The brackets are signifying putting self in a list and if you look at the add_children implementation above you see will we iterate through this list to add it to that Person's list of children.
