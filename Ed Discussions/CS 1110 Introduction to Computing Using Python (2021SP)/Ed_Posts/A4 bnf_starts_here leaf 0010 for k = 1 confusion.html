Let me first clear something up. For the k = 1 case, there are times where you would like to return False. I found the footnote on page 8 to be rather useful for figuring out these cases when I did the assignment myself. For the back-and-forth between users A and B at Post 0010 where k = 1, we should return a single element list containing just post 0010. The first error would occur if you returned a list that isn't as specified above, and the second error would occur if you returned false as there is a back-and-forth.
