Are we allowed to use methods within the class Course in the implementation of other methods? My partner and I are trying to use conflictsWith in our implementation for changeTime but are being met with an error that conflictsWith is not defined, which seems odd.

Author: Elisabeth Finkel (admin)

You are certainly allowed to write calls to Course methods outside of the Course class. Make sure you're calling conflictsWith as a method - if a and b are Course instances, the syntax is a.conflictsWith(b)
