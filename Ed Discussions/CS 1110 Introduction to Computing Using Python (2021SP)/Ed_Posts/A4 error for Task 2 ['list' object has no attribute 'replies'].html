The error says that you're doing the following 
 path = response.author + ', ' + user_paths(response.replies)
Note that the precondition for user_paths is that whatever you pass in must be a Post object. However, you're passing in a list object, because 'response' is likely your Post object and so 'response.replies' would be the list of post objects.
