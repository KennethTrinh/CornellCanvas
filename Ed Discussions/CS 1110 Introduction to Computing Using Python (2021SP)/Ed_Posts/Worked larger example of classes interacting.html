Hi all, for a larger (but prelim-sized!) worked example of classes interacting, look at problem 5 of Spring 2014 Prelim2: exam itself , solutions

Yes, that example also involved courses, schedules, and students, but they are different classes than what you are working with in A6. For example, 2014-Courses have an attribute students which is the list of students enrolled in that course. For a 2014-student, a semester Schedule on that 2014 prelim had an attribute courses tracking which Courses a Student is taking. So method add_course for a 2014-Student looks like this: 

class Student:

    """attribute schedules: list of schedules from all semester, in reverse 
    chronological order, with the schedule for the current semester being at 
    position 0 in that list."""

    def add_course(self, course):
    
        # Add this Student to the list of students in object `course`
        course.students.append(self) 
        
        # This student's current semester is self.schedules[0]
        # That semester 
        self.schedules[0].courses.apppend(course)
