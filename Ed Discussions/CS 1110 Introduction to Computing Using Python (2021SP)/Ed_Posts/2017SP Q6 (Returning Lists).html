Since we have no way to know what the id of the list is, we can only answer these questions by indicating the list contents. (Unless you're drawing the diagrams, in which case you can write the id.) But yes, the truth is that the id of the list is what is returned. So, consider the answer to that question to be shorthand for "the id of the list object whose contents are [-1,1]"
