Why does ARCH 1120 STU 520 run for 12 hours?

Lines 235-236 (and your changeTime method if it is working ) in the testing file causes this to happen: 

new_time = "MWF 8:00am-8:00pm" 
arch1120.changeTime(new_time)

> How would these courses not conflict? Femember from the handout, section 5.1: "Weâ€™ve written almost all of this class, which is used to represent both individual schedules and course rosters, for you in file schedule.py. Because of the latter application, one is allowed to add conflicting courses to an arbitrary Schedule via the add() method." (If you were building the Cornell course roster with thousands of courses, you wouldn't want to get a message for every conflicting course!)
