Each folder for a Team object has a pList attribute that is a list of ids, each id representing a Player object. Each folder for a Player has a holdings attribute which is an int. t1.plist is therefore a list of player objects t1.plist[i2] is the i2'th player in t1.plist t1.plist[i2].holdings is the value of the holdings attribute for that particular player.
