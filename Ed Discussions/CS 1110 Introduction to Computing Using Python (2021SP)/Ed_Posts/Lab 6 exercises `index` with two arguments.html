The way index works when you input a number as an argument for the second parameter, you are basically saying "start looking for the first 'L' you can find AFTER index 5". It won't find anything before index 5.

For the second one, you can see that you are not looking for the index of 's' in s (which is the variable that contains 'HeLLo WorLd!'), you are looking for the index of 's' in 's' (which is just a string, 's')
