is the assertion of isinstance() the same as the assertion of type()?

eg

is assert isinstance(n, str) the same as assert type(n) == str ?

----------- REPLIES -----------

Author: Priya Malla (admin)

Vote Count: 1

Yes, isinstance(n, type) will return a boolean so you can use it in assert statements the same way you can have boolean expressions like type(n) == str in an assert statement.
