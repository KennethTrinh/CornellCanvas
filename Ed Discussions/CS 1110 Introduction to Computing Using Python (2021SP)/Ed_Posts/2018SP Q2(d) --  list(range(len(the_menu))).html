Hi, You do not have to use list, a range loop without list would also be a correct implementation. Here is an example of printing list(range(example)))) example = ['hello', 'hi'] print(list(range(len(example)))) Using list would just list every number in the range of example. It is equivalent to a normal range loop. If you loop through this list, the for loop would iterate through every number in the list. Best, Troy
