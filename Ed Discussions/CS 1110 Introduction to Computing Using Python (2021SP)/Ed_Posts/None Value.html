If you try writing None (as you would True ) on the terminal in python, you can see also nothing shows up. Same as if you try to do x = None and then type x into the terminal. Nothing shows up either. Python in the command line just doesn't show the "None" value. Nevertheless, if you tried your method call and compared it to None, as such: no_return_method() == None , you can see you get True back, meaning that you are getting None from the function call (even if it does not show). Sidenote: If the function has no return statement, it is most of the time bad practice to try to do anything with its return value. If the function is designed not to return anything, it is probably not a good idea to try to do anything with the None value it returns.
