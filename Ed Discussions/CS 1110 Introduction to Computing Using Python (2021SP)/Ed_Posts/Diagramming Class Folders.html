Class folders have a different notation (so you can easily distinguish them from (other) object folders): "tab" on top right-hand side, name of class on the tab. (See also the study guide for some remarks on drawing class folders.) Non-class objects stay the same: tab on the top-left-hand side, id on the tab, type of the object on the upper-right. Example from Lec 17 slides of a class folder.
