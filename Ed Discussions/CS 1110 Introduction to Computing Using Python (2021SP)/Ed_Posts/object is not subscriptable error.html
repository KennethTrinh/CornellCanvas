While the lists inside a College object are subscriptable, the College object itself is not. Make sure to look at the attributes of a College object in college.py as well as the functions inside it!

(Incidentally, "Subscriptable" means, in practice terms, "accessible using square brackets". For a list, you can say mylist[0]; for an object you need to use the "dot" notation, myobject.secretname . )
