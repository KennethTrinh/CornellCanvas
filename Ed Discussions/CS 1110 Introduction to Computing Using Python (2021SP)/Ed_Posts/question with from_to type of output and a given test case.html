Your assumption isn't right. Unless you convert a string to an int after string slicing, when you are string slicing (even if the string you are supposed to get back is a number), you will get back a string, not an int. In the example you gave, nowhere in the test is zero an integer. Zero is either a string by itself, or part of a string.
