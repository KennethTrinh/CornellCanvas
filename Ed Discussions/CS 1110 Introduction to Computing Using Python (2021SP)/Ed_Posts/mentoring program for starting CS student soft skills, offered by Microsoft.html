Dear students,
Want a mentor in Tech? Whether you’re taking CS because you want to dip a toe in the tech waters or because you are fully committed to a tech journey, a new Microsoft mentoring program has been developed with you in mind. The goal is to provide additional support for starting CS students focusing on non-tech (soft) skills including problem solving, resilience, interpersonal skills, growth mindset, etc.
During this program, Microsoft employees will serve as mentors, coaching your cohort of peers through group and individual remote sessions. You’ll work on developing individual and collaborative strategies to help navigate challenges that can arise across personal, academic and workspaces.
The spring mentoring program will launch late March 2021. You’ll engage in five weekly group sessions (remotely) with Microsoft mentors coaching you through discussions and embedded tech activities, and, have an opportunity for two 1:1’s.
NOTE:
This is not a program designed to teach Microsoft tools, technologies or programming.
You will not be talking about resumes or careers at Microsoft. Mentors are unable to recommend you for internships or positions. This program is not designed to be a recruiting mechanism for Microsoft.
Space is limited, so we cannot guarantee your participation. If admitted, you’ll be notified in advance of the start date.
If you cannot commit to full participation, we ask that you do not sign up. Attending all sessions in full is mandatory.
Please complete and submit this MS Mentoring Program application form by March 5, 2021 at 5:00pm PT for consideration. The Microsoft Mentoring program team will review and admit students on a rolling basis starting January 9th.
