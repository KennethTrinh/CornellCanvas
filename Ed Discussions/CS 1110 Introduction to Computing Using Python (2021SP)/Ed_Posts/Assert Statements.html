If you are asked to do so, you should be able to do so; but if we don't ask for them, don't waste time writing them on the exam.

Form: assert <some boolean condition>

or (I prefer) assert <some boolean condition>, <string that's a message to print out if the boolean condition is false> If the boolean condition is true, nothing happens. If the boolean condition is false, execution is stopped with an error and if you give the string message, that's printed out. You can use assert statements for purposes other than class definitions, too --- check out the testing code for a4!
