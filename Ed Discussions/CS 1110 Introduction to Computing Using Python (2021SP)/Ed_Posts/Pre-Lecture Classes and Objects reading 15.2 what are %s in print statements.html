Oh dang! We haven't learned '%' for print statements yet (and probably won't bother to do so in CS1110, since it's very Python-specific)! View that line as the equivalent of string concatenation in the way we're used to, in the sense demonstrated by the following example code. (Hit the "Run" button and marvel at Ed Discussion's power...) first = 100.1 second = 200.2 # what you're used to: print("Output using string processing we are used to:") print('(' + str(first) + ', ' + str(second) + ')') # what the print statement with the %s does print("Demonstrating that the % syntax you don't need to know produces the same output") print('(%g, %g)' % (first, second))
