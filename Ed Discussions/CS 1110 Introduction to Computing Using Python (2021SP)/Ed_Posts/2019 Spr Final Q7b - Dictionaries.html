Hi! Let me know if I'm misunderstanding your question, but: it might help to think of a dictionary as just a fancy list where we have keys instead of indices 0, 1, 2 and so on. If self.goods had been a list , then setting self.goods[i] to p would make the entry self.goods [i] the entire object p , not just p.name . Analogously, because self.goods is a dictionary, setting self.good[p.name] to p puts p in entry p.name of the dictionary self.goods . class Example: def __init__(self,n): self.name = n def __repr__(self): return "Example named " + str(self.name) goods = {"iPad": Example("iPad"), "Surface": Example("Surface")} print(goods) goods["iPad"] = Example("iPadInfinity") print(goods)
