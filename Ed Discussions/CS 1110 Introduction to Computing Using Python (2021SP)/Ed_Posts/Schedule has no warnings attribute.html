Something seems strange, because we already provided you with an _ _ init _ _ method for Schedule tjat creates the warnings attribute. Do you have these lines in your schedule.py? (For me they are lines 30-40, indented under the class Schedule header? 
    def __init__(self, n):
        """Initializes a new Schedule with name `n`, 
        attribute `courses` to the empty dictionary {},
        and attribute `warnings` set to the empty list.

        Preconditions: 
            n: non-empty string.
        """ 
        self.name = n 
        self.courses = {} 
        self.warnings = []
