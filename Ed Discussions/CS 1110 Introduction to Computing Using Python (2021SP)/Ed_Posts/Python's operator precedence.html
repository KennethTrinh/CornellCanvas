-2**3 is same as -2^3.

This is equivalent to cubing 2 and then negating your answer. It is different from (-2)**3 or (-2)^3. In this case, the parenthesis takes precedence. Hence the solution would be cubing -2

Good question! Here's some expressions that demonstrate: 

Python 3.8.5 (default, Sep  4 2020, 02:22:02)
[Clang 10.0.0 ] :: Anaconda, Inc. on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> -(2**2)
-4
>>> (-2)**2
4
>>> -2**2
-4

So you can see that Python treats -2**2 as if there were invisible parentheses around the "2**2" part, i.e., the exponentiation gets done before the unary-minus gets applied.
