While I'm not Professor Lee, I think that the hint to solving that problem is in Section 7.1.1 of the assignment description document:

"A back-and-forth of length k between user u1 and (different) user u2 starts at a post P if (and only if) P was written by u1 and ...

• Case: k = 1: ... there is no reply to P written by u2. (This includes the case where P has no replies at all). [14]

• Case: k ≥ 2: ... there is a back-and-forth of length k − 1 between u2 and u1 starting at some reply to P.

[14]: Contrariwise, no back-and-forth of length k = 1 between u1 and u2 starts at P if P wasn’t written by u1 or if there’s a reply to P written by u2."
