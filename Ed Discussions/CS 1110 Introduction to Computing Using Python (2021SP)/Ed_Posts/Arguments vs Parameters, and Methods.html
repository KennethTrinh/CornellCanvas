Arguments are the values given to a function as input. Parameters are the variables where the function stores the values given as input. You can tell P is a method call by the fact that it's called by "<some object>.<method_name>". A method can only be called by including an object that has access to that method and then a "dot", and then the method name. This makes sense because "join" is designed to work with respect to a specific string, the one that is going to be the "joiner".
