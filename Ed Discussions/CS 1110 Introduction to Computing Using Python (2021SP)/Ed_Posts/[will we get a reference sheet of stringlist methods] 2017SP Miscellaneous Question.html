Decision: we will not be pre-announcing the contents of any function/method specification list. You will be expected to know the syntax and specification for list and string slicing (the [:] notation, the index() method for strings and lists, and the append() method for strings without the aid of a reference sheet. But we will provide specifications for methods/functions/operators that we don't expect you would already know from doing the assignments and labs.
