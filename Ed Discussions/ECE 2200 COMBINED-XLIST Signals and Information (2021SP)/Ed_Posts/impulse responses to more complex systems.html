If we were given a causal non-fir filter $$y[n] = b_kx[n-k] + ... + b_0x[n] + f[n]$$ where f[n] has no relation to the input x[n], could you find the impulse response by plugging $$\delta[n]$$ into y[n] ignoring the f[n], and then adding on f[n] to h[n]? If this is the case, and you want to find the output response for some x[n] input, how would you efficiently perform $$x[n] \circledast h[n]$$ when h[n] is no longer a sum of delta functions? Actually, the system you specify above isn't time-invariant, so all bets are off. The LTI non-FIR filters we'll discuss take this form: $y [ n ] = \sum _ {k = 1} ^ N a _ k y [ n - k ] + \sum _ {k = 0} ^ M b _ k x [ n - k ]$. The "recursive terms" on the RHS are what make the system non-FIR. For these systems, the impulse response in general has infinite duration. Thus it's an "infinite sum of $\delta$-functions," if you will.
