The way we did it in class: say the FIR system's I/O relationship is $y [ n ] = \sum _ {k = 0} ^ M b _ k x [ n - k ]$. Let the input be a $\delta$ and the output that arises we call the system's impulse response and denote it $h [ n ]$. $h [ n ] = \sum _ {k = 0} ^ M b _ k \delta [ n - k ]$. For $n < 0$ and $n > M$, all terms on the right are zero. When $0 \leq n \leq M$, the $k = n$-term is $b _ n$ and the rest are zero. Accordingly $h [ n ] = \left \{ \begin{array}{cl} 0 & {\rm when} \; n < 0 \\ b _ n & {\rm when} \; 0 \leq n \leq M \\ 0 & {\rm when} \; n > M \; . \end{array} \right .$
