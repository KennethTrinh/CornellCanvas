Dunno why that's missing. Answer is as follows. From the spectrum, $x ( t ) = 3 + 2 \cos ( 2 \pi ( 2 ) t - \pi / 2 ) + \sqrt{2} \cos ( 2 \pi ( 2 ) t + \pi / 4 )$. Thus $x ( 0 ) = 3 + 0 + \sqrt{2} ( 1 / \sqrt{2} ) = 4$, $x ( t )$ is periodic because the frequencies of the terms are harmonically related, and because the fundamental frequency is 1 (the gcid of 2 and 5), we have $x ( 17 ) = x ( 3 )$.
