Parts (a) and (b) ask you to find impulse responses of various systems. To find the impulse response of a system, you set its input signal to be $\delta [ n ]$ and find what the output signal is. In (b), you set the input of the first system to be $\delta [ n ]$ and find the output of the overall system. FYI we didn't quite get to cascaded systems in today's lecture, which I'd hoped we would, but you can actually do this problem using stuff we've covered already.

I'm not sure Ed is the best way for me to address these questions beyond what I wrote above. I would just say, for (b), set $x [ n ] = \delta [ n ]$ and figure out what $y [ n ]$ comes out to be --- that'll be $h [ n ]$, what the problem asks for. The impulse response is the response to an impulse --- that's the definition. It's also what you convolve with inputs to give outputs, but for this problem I recommend thinking of it in the primary way.
