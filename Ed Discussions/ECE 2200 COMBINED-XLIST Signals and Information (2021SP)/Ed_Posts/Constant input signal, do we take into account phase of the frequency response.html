Title: Constant input signal, do we take into account phase of the frequency response

If I have a constant input signal x[n]=1, and the frequency response at 0 is H\left(0\right)=1\cdot e^{j\varphi} for some non-zero phase, is the output signal y[n]=1 , or y[n] = 1*cos(phi). Basically do we take into account phase in the output signal for constant signals?

----------- REPLIES -----------

Author: David Delchamps (admin)

Vote Count: 1

If the system has a real-valued impulse response $h [ n ]$, then $\widehat{H} ( 0 )$ will always be a real number. However, in general, constant input signal $x [ n ] = 1$ will yield constant output signal $y [ n ] = \widehat{H} ( 0 )$, whether that number is real or complex.
