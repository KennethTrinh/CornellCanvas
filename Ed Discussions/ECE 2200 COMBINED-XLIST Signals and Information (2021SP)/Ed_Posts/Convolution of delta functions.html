Is it correct to say that delta[n-a] * delta[n-b] = delta[n-a-b]?

----------- REPLIES -----------

Author: David Delchamps (admin)

Vote Count: 1

That's correct. Let $x [ n ] =$ that convolution. Then

$x [ n ] = \sum _ {k = - \infty} ^ \infty \delta [ k - a ] \delta [ n - k - b ]$.

The ostensibly infinite sum on the right-hand side has exactly one nonzero term, the one where $k = a$, so we get

$x [ n ] = \delta [ n - a - b ]$.
