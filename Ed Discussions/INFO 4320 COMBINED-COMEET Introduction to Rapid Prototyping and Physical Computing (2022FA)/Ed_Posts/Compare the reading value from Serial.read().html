When you input text in Serial Monitor, the input data is transmitted as char or String coded using ASCII Codes: https://www.computerhope.com/jargon/a/ascii.htm#:~:text=Short%20for%20American%20Standard%20Code,the%20language%20of%20all%20computers. If your input is '1', the 
    incoming_byte = Serial.read();
value of incoming_byte is actually 49, which is the ASCII code for '1'. If you want to check whether the incoming_byte is '1', you need to use one of the code below: 1. if (incoming_byte == '1') 2. if (incoming_byte == 49) Both should work.
