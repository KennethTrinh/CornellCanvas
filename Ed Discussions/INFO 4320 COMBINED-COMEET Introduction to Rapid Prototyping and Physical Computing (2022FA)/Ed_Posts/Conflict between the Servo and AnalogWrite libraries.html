Title: Conflict between the Servo and AnalogWrite libraries

Author: Ruidong Zhang (admin)

Some of you met issues while trying to use the servo and LEDs, buzzers, or DC motor together. This issue is caused by conflict in the Servo and AnalogWrite libraries. There is a workaround to this issue - not using these two libraries and use the lower-level PWM library ledc instead. Here is a detailed tutorial on how to use ledc instead of analogWrite to control LEDs. It can be used to control buzzers too. Here is a detailed tutorial on how to use ledc instead of Servo to control servos. If you meet with similar issues, this should solve your problem.
