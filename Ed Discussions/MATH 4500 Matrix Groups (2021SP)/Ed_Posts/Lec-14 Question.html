Recall the definition: the path component of $I$ consists of all $h\in H$ for which there exists a path $\delta\colon[0,1]\to H$ with $\delta(0)=I$, $\delta(1)=h$. $\gamma_B$ is a path in $H$ which starts at $I$ and ends at $B$. Therefore $B$ is in the path component of $I$, which was to be proved. You are right that it then follows that $\gamma_B(t)\in H_0$ for all $t$.
