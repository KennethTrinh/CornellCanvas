No: $A=SBS^{-1}$ does not imply $\|A\|=\|B\|$ (Hilbert-Schmidt norm).

Recall $\|A\|^2=\operatorname{trace}(A^*A)$. If $A=SBS^{-1}$, then $A^*A=(S^{-1})^*B^*S^*SBS^{-1}$, so using $\operatorname{trace}(AB)=\operatorname{trace}(BA)$ we get

$$\|A\|^2=\operatorname{trace}(A^*A)=\operatorname{trace}((S^{-1})^*B^*S^*SBS^{-1})= \operatorname{trace}(B^*S^* SB(S^*S )^{-1}).$$

So $\|A\|=\|B\|$ if $S$ is unitary ($S^*S=I$), but not for general invertible $S$, not even if $B$ is diagonal.

For instance, if $B=\begin{pmatrix}\lambda_1&\\&\lambda_2\end{pmatrix}$ and $S=\begin{pmatrix}1&1\\&1\end{pmatrix}$, then $A=\begin{pmatrix}\lambda_1&-\lambda_1+\lambda_2\\&\lambda_2\end{pmatrix}$, so $\|B\|^2=|\lambda_1|^2+|\lambda_2|^2$, whereas $\|A\|^2=|\lambda_1|^2+|\lambda_1-\lambda_2|^2+|\lambda_2|^2$.
