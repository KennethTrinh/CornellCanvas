Title: How to make the Logic Analyzer Work For Windows

Author: James Lepone (admin)

Date: 2022-02-15T08:18:54.370736+11:00

Category: General

Vote Count: 3

1. Download and install PulseView 0.4.2

2. Open Zadig (this was automatically installed with PulseView)

3. Select WinUSB (should be default) and press "Install Driver"

4. After install, close Zadig. Then, open PulseView.

5. Follow the slides to search for the logic analyzer. It should now show up as Salae Logic.

Note: It might take closing and reopening PulseView one or two times before it is detected.
