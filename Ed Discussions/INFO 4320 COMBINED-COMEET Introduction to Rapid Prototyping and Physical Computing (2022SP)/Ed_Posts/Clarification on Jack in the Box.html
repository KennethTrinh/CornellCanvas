Some clarification on the second design assignment:

A more apt name would be "interactive laser-cut box project". It DOES NOT have to be a traditional jack-in-the-box with a spring/crank. You are to design and build a laser-cut box with a moving component. This component should be powered by a motor, which should be inside the box. The trigger for this mechanism does not have to be the box opening--you can use any sensor in your kit.

A common method to build the box is to modify the design from the Advanced 3d Design slides (fourth lesson). The slides have a link to the makercase box files , which can be downloaded and imported into Fusion 360.
