Author: Ruidong Zhang (admin)

The Y and Z axes are actually further behind the X axis. This is probably because you have some kind of delay between SPI requests. E.g., if you send the readings to LCD immediately after getting the readings, you will see that XYZ axes are separate, because LCD operations take time. This is OK. If you want three axes to be seen in the same window, you can try requesting XYZ axes data consecutively and put all LCD parts of the code after getting data from all axes.
