Yes. You need to use the laser cutter for these parts.

This is the detailed requirement about the assignment on canvas:

  • The box and all the mechanisms must be built using a laser cutter (but you can use gears and axles from your Lego kit);

  • The animation mechanism should be inside the box;

  • The animation mechanism must use the motor, the servo, or the stepper provided in the kit;

  • The system must be controlled by your ESP32;

  • The mechanism might be triggered by opening the box or detecting an attempt to interact with the box;
