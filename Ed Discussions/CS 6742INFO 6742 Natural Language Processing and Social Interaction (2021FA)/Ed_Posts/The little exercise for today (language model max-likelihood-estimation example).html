Recall from the blackboard (where $C_L$ is some normalization constant with factorials and stuff involving $L$ and the $S_i$'s that you can ignore and I can never remember): C_L\prod_{i} \phi_i^{S_i} + \lambda\left(\sum_k \phi_k - 1\right) Calculus tells us that if we're trying to find the arg-max, we should "set the derivative to zero and solve for the unknown". What is the unknown, that is, what variable(s) should we take the derivative with respect to? (Answer is below) Try solving for the unknown. Does using the log of the lefthand side make things easier? (Hint/motivation: you'll often see papers working with the log ) (Answer to 1 above: The $\phi$ variables are the unknowns. I find it easiest, in order to keep all the notation and variables straight, to use, for each j , : \frac{d}{d\phi_j} That is, use a different variable, j , than the index variable i for the product. )
