    It seemed like these paper counts probably constitute more than ten unique authors. Other researchers have tackled this problem "https://arxiv.org/abs/2103.07534" and it seems pretty difficult. I have worked out two current decent solutions that don't require an entirely separate research project. The first is to add the major category ( eg 'cs', 'dg-ga', 'econ', 'eess') to the author name to create a separate author for each instance of a name-field. This probably represents a way in which authors choose their own writing names, but doesn't reflect the fact that many will post in more than one major category. The second option is to add affiliations in. Using another data source, I was able to do this for a limited set relatively easily, but it seems like it would disadvantage grad students in my analysis (and the analysis is likely heavily dependent on them). I'm a little stuck on this part although I think the first option is workable. If I could use google scholar, I might be able to do a better job, but as far as I can tell making this many searches would get me IP banned or somehow otherwise get my requests rejected.

    Name disambiguation: the google scholar option wouldn't see workable to me even if one had the network bandwidth: they are also not that great at disambiguating names (as is nobody, since it's hard.) An option I think is fine for this class project and its hypotheses is to scrape a list of grad student and faculty names names (say from some institution webpages) and select relatively unique names. This is very obviously a highly biased sample, but you want to start somewhere with your hypothesis. (If you saw anything promising, you would then say, "OK, it's worth trying to do name disambiguation"; if not, you saved yourself a lot of trouble.) Re: fightin' words: the "newer" words look surprisingly generic, and vice versa for the "old". Might be good to do a first check on where 'stupidity' and 'stumble' and 'studio' are happening, and also do a fightin' words on two different (sub)categories, for debugging. (Also, is "nonseperable" a typo or is the mis-spelling apparently indicative of date?)
