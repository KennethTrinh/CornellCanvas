Hi! My group and I are working with the Slashdot data and wrote some clean-up code so that we could use Python's XML parser (xml.etree). Attaching code & instructions, if it might help you as well: What the script does: Fixes the <tokenext> tag --> <tokentext> . Places a parent <root> tag around the whole file (so <root> in the beginning, </root> at the end). Removes all <htmltext> ... </htmltext> tags. (We did this because we didn't need the information included in it, and it was complicating the XML parsing because html formatting within it were not always consistent.) Removes all non-UTF8 characters. To use: Download the slashdot data Download the attached files and copy them into the data folder you want to clean up (e.g. into slashdot_part_1 ) so that the folder now has all the *instancedata.txt , clean_data.sh , and remove_htmltext.py files. Run the script: . ./clean_data.sh on Mac.
