Any course that says " department consent required " is currently being managed by the student services office via an internal waitlist. If you are interested in one of the courses and would like to be put on a waitlist please email ugrad-coordinator@ece.cornell.edu with the following information: Course code Which discussion/lecture you want You can Cc the faculty of the course. Please be aware that some of these waitlists are already very long, so we recommend having an alternative course. There is no need to ask about your spot on the waitlist, we will share enrollment access with students when we have it.

The root problem here is that Peoplesoft, the database system that forms the backend for studentcenter, does not have a satisfactory waitlist system. So we need to manually effect waitlist functionality by requiring department consent to enroll in a class that has filled up and then emailing PINs to students as spots become available.
