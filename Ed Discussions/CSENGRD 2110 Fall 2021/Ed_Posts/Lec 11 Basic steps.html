A mistake, which is of no real consequence. Sorry about that. Below is what slide 16 says. We should have written (C+2) + k basic steps on slide 17.

You can make the rest of the changes on line 17 and see that it doesn't make two much difference.

=======================

1.Create new String object, say C basic steps.

2.Copy k chars from object s to the new object: k basic steps

3.Place char ‘c’ into the new object: 1 basic step.

4.Store pointer to new object into s: 1 basic step.

Total of (C+1) + k + 1 basic steps.

------------------------------------
