A class's package is determined by the package line at the top of the file, so to put class A3 in package a3 , you would write package a3; as the first line of "A3.java". It should also go in a folder named "a3". There are many ways to do this graphically in Eclipse as well, such as creating a new package and dragging the files into it. But ultimately all that matters is the package line in the source code (and being in the right folder can be important when compiling larger projects).
