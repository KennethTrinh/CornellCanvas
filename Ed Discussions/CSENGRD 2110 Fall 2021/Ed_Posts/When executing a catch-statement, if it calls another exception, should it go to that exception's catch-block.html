Exceptions are thrown , not called (though they can be constructed). When an exception is thrown, it will be handled by the catch block associated with the nearest enclosing try block. A try block does not enclose its own catch blocks. Exceptions can be (re-)thrown in a catch block. Such exceptions would have to be handled by a different try/catch statement higher in the call stack.
