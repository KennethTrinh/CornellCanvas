Author: David Gries (admin)

Vote Count: 0

If the elements to be inserted or removed are clustered near the end of the ArrayList, then it doesn't take O(n) time to insert or add. To remove an element 2 from the end requires moving on two elements down in the ArrayList. In order to remove element i from a linked list, you have to search through I-1 elements to get to it. If that i is close to the size of the list, n, that takes time O(n).
