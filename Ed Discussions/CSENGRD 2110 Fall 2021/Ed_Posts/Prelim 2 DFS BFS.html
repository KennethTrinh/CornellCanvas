We wrote two recursive dos methods in class. You should be able to develop them yourself, using high-level statements as done in lecture.

You should be able to write the iterative versions of DFS and BFS as well. Just know the loop invariant, e.g. for DFS:

Every node that is to be marked a visited is reachable along an unvisited path of nodes from some node on the stack.
