Author: David Gries (admin)

By the Compile-time Reference Rule (PLEASE LEARN IT!), from a variable of (static) type Animal, one cannot reference methods that are declared only in Cat. However, if you know the object pointed at has a Cat partition, you can cast it down to Cat and then reference the methods.
