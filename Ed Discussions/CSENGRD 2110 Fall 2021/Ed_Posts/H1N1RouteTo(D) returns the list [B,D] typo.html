In the method spec of H1N1RouteTo it says that H1N1RouteTo(D) returns the list [B,D]. If I'm understanding correctly, wouldn't it return the list [A,B,D]?

The method spec says: B.H1N1RouteTo(D) returns the list [B,D] , which is correct, as it is giving the route from B to D. If the call was: A.H1N1RouteTo(D) , then the returned value would be the list [A, B, D] .
