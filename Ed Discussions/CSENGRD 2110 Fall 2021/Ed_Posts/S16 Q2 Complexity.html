The operations remove and add are sequential in this case. You first remove, which takes O(n) time, and then you add, which takes O(n) time. So list.add(0, list.remove(i)) takes O(n) time. Remember when you add, it's O(n) because prepending shifts all the elements down the array, but we're not removing an element each time we shift. We remove, then we add.
