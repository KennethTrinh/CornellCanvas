An expression is evaluated to yield a value. A statement doesn't yield a value. The statement System.output.println(5); does not give a value, although it prints something out. Therefore, this is not a legal expression: 4 + System.output.println(5); A while-loop is a statement to be executed: while (x < 5) { s= s + x; x= x+1}; The distinction get blurred in Java because the assignment statement x= x+1; written without the semicolon, is actually defined to be an expression; its value is the value assigned to x. Therefore, one can write this: y= 5 + (x= x+1); This adds 1 to x and then stores in y the sum of 5 and the new value of x. We NEVER writes such things because it is easy to misunderstand.
