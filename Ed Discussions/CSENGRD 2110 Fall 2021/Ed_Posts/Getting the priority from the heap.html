Just which part are you trying to write? The answer depends on what part you are trying to write.

If f is in the frontier set, then d[f] contains the shortest distance from the start node to f, using only paths that contain red nodes except for f itself.

If there is an edge (f, w) and a node w is being moved from the far-off set to the frontier set, then w's shortest distance has to be calculated. It is d[f] plus the weight of the edge (f, w), since the path starts at the start node, winds up in f, and then hops along (f, w) to w.
