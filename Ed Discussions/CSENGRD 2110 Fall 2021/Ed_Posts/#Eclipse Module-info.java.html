Author: David Gries (admin)

Category: General

Vote Count: 0

Problem:

In the code you have this error: "Must declare a named package because this compilation unit is associated with the named module 'module'"

When you run you get this error:

"Error occurred during initialization of boot layer java.lang.module.FindException: Error reading module: <filepath> Caused by: java.lang.module.InvalidModuleDescriptorException: ClassName.class found in top-level directory (unnamed package not allowed in module)"

Solution:

In your package explorer, find module-info.java and delete it

Prevention:

When you create a new Java project, Eclipse will ask you if you want to create module-info.java:

Select "Don't Create" and you will not have this issue.
