The maximum length m refers to the longest path from the root to a leaf. In this example, it is 3, for example the string "inn" fits this. As you can see, the number n of strings have nothing to do with the answer.

When searching for a string of length k that is in the trie, well, it takes time O(k) to see that it is in.

When searching for a string that is not in the trie, well, at most one would have to look at m branches.

Hence the time min(m, k).
