Read the assignment carefully. It is explained there.

There is no such thing as an "empty node".

Consider an array length 8 containing a heap of size 5. Here is an example: (5 , 7, 6, 9, 8, ?, ?, ?) The heap is in the first 5 elements: 5, 7, 6, 9, 8. The rest of the array element contain values, but we don't care what they are.

In such a notation, ? represents a value in the array that is not in the heap, and we don't care what the value is.
