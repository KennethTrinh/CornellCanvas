Author: Varun Gande (student)

b has static and dynamic type B ob has static type Object and dynamic type B d has static type D and dynamic type B

Each static type has a method toString in its class or a superclass, so CTRR says the method call is legal.

Now, use the dynamic type to determine which version of method toString is called. All variables have dynamic type B, so B’s version is called by the bottom-up rule.

The restriction depends on dynamic type only. If you made a new Object (e.g. Object x = new Object()) and invoked x.toString(), that would use Object’s toString method. It’s all according to the bottom-up rule.
