I'm pretty sure the answer given is correct.

F(3) is called first. Since 3 is not equal to 0 or 1 you evaluate F(n-1) + F(n-2). Call F(n-1) where n = 3, which is F(2) , first. Since 2 is not equal to 0 or 1, then evaluate F(n - 1) where n = 2 first, which is F(1) . 1 equals 1, so F(1) returns 1. Then you evaluate F(n - 2) where n = 2 which is F(0) . F(0) returns 1. Then 1+1 is returned by the call of F(2). Now call F(n-2), where n = 3, which is F(1) . That returns 1, so F(3) returns 2 + 1, which is 3.

Not quite, remember that a full method call must terminate before the code that called said method can continue. This means F(2) must finish executing before F(n - 2) where n = 3 can be called. For F(2) to finish executing, both F(n - 1) where n = 2 an F(n - 2) where n = 2 must be called (i.e. F(1) and then F(0))
