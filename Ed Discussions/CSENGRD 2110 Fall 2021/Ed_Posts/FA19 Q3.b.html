Apologies, the use of .equals() to compare the values of maxDepth (an int ) is a typo. It is impossible to invoke methods on primitive types (they do not inherit from Object ), and autoboxing will not help us here (this is easy to verify using JShell). Equivalence tests of primitive fields must always be done using == .
