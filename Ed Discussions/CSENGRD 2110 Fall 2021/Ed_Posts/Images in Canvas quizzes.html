Title: Images in Canvas quizzes

Author: David Gries (admin)

Date: 2021-10-17T20:57:34.876562+11:00

Category: General

Vote Count: 0

Some Canvas quizzes will have embedded images --e.g. pictures of inheritance diagrams. Mac users beware: there is a known bug in the interaction between Canvas and Safari, which can cause Safari not to display embedded images. The fix is easy: use Chrome instead of Safari for the quiz. We have confirmed this with Cornell IT. If you did somehow start taking the quiz in Safari and did run into this problem, you can work around it in a way that doesn't lose any significant time on the quiz: right click on the broken image icon, save the image, and open it up in Preview. But we recommend using Chrome anyway to avoid even this small hassle.
