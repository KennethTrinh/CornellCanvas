Yes, for each node in the settled set, there is a shortest path from the start node to that node that consists only of spelled node. The algorithm as developed calculates on only the distances of shortest paths, not the shortest paths themselves. The lecture on Thursday will show how to extend the algorithm to compute the shortest the paths.
