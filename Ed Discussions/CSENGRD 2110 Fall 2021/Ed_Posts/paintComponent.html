Author: Curran Muhlberger (admin)

paintComponent() is called by the Swing framework whenever it determines that the component needs to be redrawn. Usually this is either when the window is resized or when an event handler calls repaint() . If you have an unconditional println() in your paintComponent() method and you resize the window, you should see its message printed to the console. If not, it is possible you are not running the code that you think you're editing (a common mistake).
