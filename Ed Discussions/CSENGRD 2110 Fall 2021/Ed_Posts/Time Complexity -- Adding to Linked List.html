Think back to your linked list project. To add to that linked list, all you did was change the values of some of the pointers of the nodes on either side of the node you were adding as well as assign values to the node you were adding. The same is the case for any LinkedList, so adding an element to a linked list doesn't depend on the size of the list. That means it has an average time complexity of O(1).
