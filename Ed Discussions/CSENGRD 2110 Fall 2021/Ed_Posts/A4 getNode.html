Hi! For getNode(int h), the precondition mentions that 0<=h<size of list. If the list is empty, size would be 0, should getNode(int h) just return null? Thank you very much!

----------- REPLIES -----------

Author: David Gries (admin)

Vote Count: 1

If size is 0, then 0 <= h < size is impossible and the call to getNode does not satisfy the precondition.

Therefore, your method can do anything it wants. You don't have to handle this case.
