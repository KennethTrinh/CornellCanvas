I was working on this for awhile and I think I figured it out, or at least something that works. No idea where the directions referenced in #190 are. Extract the A2Skeleton.zip file into a file called a2. You can extract it into your eclipse workspace file, wherever that is. It contains A2.java and A2Test.java. In Eclipse, go to your Package Explorer pane and right click to create a new project. Name it a2, and keep "Use default location" checked IF you put the a2 file into your eclipse workspace (otherwise, uncheck it and browse to where you put a2). Click Finish, and java will make a project using the A2 and A2Test files. Right click on the project file in your Package Explorer. Go to Properties > Java Build Path > Libraries. Click on Classpath, then do Add Library and add JUnit 5 to the Classpath. Click Apply and Close. That should make it so you're able to test. Hope this helps.
