"this" can never be null. Look that up in JavaHperText and you will see that it is defined as follows: this is an expression; it evaluates to the name of (pointer to) the object in which it appears. Thus, if it appears in an object, that object must exist, so there is a pointer to it. Try putting something like "this.name" in a place where it is not in an object (example, in a static method; you will learn about "static" VERY soon), and you will see that it gives a syntax error and the class will not be compiled.
