getNames(sp) iterates through sp (in this case route4) to Return the names of Persons in sp, separated by ", " and delimited by [ ]. Since route4 is null, this throws a NullPointerException when you try to iterate through it (at the line: for (Person p : sp) { ). You could do the test without using getNames , as described below.
