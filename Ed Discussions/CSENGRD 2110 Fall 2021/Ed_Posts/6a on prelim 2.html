I cannot understand why the following code would not work for 6a on the prelim: /** Visit every node reachable along a path of unvisited nodes from node u. Precondition: u is not visited. */ public static void dfs(Node u) { if(u has been visited) return; Visit u; for each neighbor w of u: dfs(w); } The precondition guarantees that u has not been visited. So then the next line allows it to be visited, and after that, recursion allows the same process to take place for each neighbor until all nodes that are reachable have been visited. I know my logic in this must be wrong somewhere I just can't figure it out

Author: David Gries (admin)

One must go by the specification!!! The specification says that u is not visited. Therefore, if a neighbor w is already visited, dfs(w) should not be called.
