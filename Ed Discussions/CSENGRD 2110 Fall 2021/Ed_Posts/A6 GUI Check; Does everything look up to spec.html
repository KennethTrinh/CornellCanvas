Your GUI looks great! The stylistic differences are expected on a Mac - the default "look-and-feel" of Swing on Macs is one that tries to emulate native applications, whereas on Windows and Linux, the default look-and-feel is the cross-platform "Ocean/Metal LaF". It is possible to change the look-and-feel in code (see the Swing Tutorial, or the code in SwingSet2), but for this assignment, the default is fine. Good work!
