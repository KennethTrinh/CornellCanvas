Author: David Gries (admin)

Vote Count: 0

There IS not declaration of y in the body. Please use correct terminology.

this.y= y; is NOT a declaration of y, it is simply an assignment.

We prefer that the assert statement be placed BEFORE this assignment statement. But note that the assert statement must use parameter y and not field this.y.
