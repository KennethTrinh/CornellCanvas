Author: Varun Gande (student)

Vote Count: 0

If the static type of the variable is Animal, it cannot access methods ONLY declared in Cat. For example if you have Animal an = new Cat();, you can access methods in Cat that are declared in Animal or a superclass of Animal (by CTRR), but you cannot access methods only declared in Cat.

Author: Varun Gande (student)

Vote Count: 0

In that case, that’s something like Animal an = new Animal(); an = new Cat(); if i’m understanding correctly.

No, you still can’t access Cat specific methods since you haven’t changed the static type. To access Cat-specific methods, you would then have to downcast an to Cat and store that pointer in a new variable—so Cat an_cat = (Cat) an;—then you can access cat specific methods using an_cat.

Search up CTRR in javahypertext for specific examples.
