The comment

// c= larger child of h (return if h has no children)

is implemented by the following three if-statements. 1. h is set to the index left child. 2. If h > k-1, then h > the index of the last element of the heap. No further bubbling needs to be done, so return. 3. If the right child (as well as the left child) exists and its value if bigger than the value of the left child, set c to the index of the right child.

You should be able to follow the rest.
