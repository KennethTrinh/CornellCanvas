A Node in an H1N1Tree object. Each H1N1Tree object contains a Person. The Person's in the Nodes of an H1N1Tree are all different.

Since the Person's in the Nodes of an H1N1Tree are all different, we often abbreviate a long phrase by just saying Node P1 (if P1 is a Person) meaning

The node who Person field contains Person P1.

I hope this helps.
