If you look on slide 4 of Prof Muhlberger's slides, you will see this: Java designers decided on this Principle: primitive operations on type int should yield an int. Another way to find this out is to use JavaHyperText. Type primitive type into the JavaHyperText field. One entry will remain. Read the 2-page pdf file. I suggest you do this anyway, even if your question is answered, because this pdf file contains valuable info about primitive types.
