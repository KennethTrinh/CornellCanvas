Try this other way. What I normally do is import the files instead of copying them over, but you'll have to know where your files are being downloaded.

TL;DR: It looks like your .jar file is in Vincent Duah > eclipse-workspace > A5. You should Select Folder A5 when configuring your ClassPath

When you create your new Java Project.

  • Left click the folder named "A5" (or name of your Java project folder).

  • Click "import"

  • Select "FileSystem"

  • On the top right is "Browse". Locate where you've downloaded the assignment files

  • Make sure "Into Folder:" has "A5" (or name of your Java project folder)

  • If you've downloaded the files to the Downloads folder in Windows, click "a5skeleton" and then "Select Folder" at the bottom left.

  • Check mark "a5skeleton" in the window on the left within the Import Window

  • Follow the steps here to configure your ClassPath http://mathcenter.oxford.emory.edu/site/cs170/externalLib/
