The picture shows three possibilities. 1. There is an edge from f to w1 in the settled set. 2. There is an edge from f to w2 in frontier set. 3. There is an edge from f to w3 in the far-off set. You have to figure out how to handle each of these cases. Hint: (1) and (2) can be handled the same way.
