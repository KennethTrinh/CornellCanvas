f(n,m) is just a function that has two parameters.

f(n,m) is:

\sum_{h=0}^n\log\left(m^{\left(h\right)}\right)

Then, how is the time complexity of the function determined?

------------------------------------

Author: Simon Huang (admin)

Vote Count: 0

so this function is basically:

= log(m^0) + log(m^1) + ... + log(m^n)

= 0 logm + 1 logm + 2 logm + ... n logm

= (1 + 2 + ... + n) logm

= (1 + n) n / 2 * logm

which is O(n^2 logm)
