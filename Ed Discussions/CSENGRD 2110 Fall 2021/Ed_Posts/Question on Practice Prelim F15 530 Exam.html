Consider Dog d= (Dog) a;

At compile-time, only the (static) TYPE of a , and NOT it's value, is used to determine whether the assignment is legal.

The type of expression a is Animal. At runtime, it is POSSIBLE for an object of type Animal to have a Dog partition. Therefore, casting to Dog is allowed.

Note that if the statement is

Dog d= (Dog) "st";

Then the cast won't compile because it is impossible for a String object to have a Dog partition.
