Author: David Gries (admin)

Vote Count: 1

Yes, given I >= 0, one has to search from the head of the list (or the end of list) for node number I, and that takes time proportional to i.

Please do not use "ith". When i is 0, what does "0th" mean? What does "1th" mean?

The first element is element number 0, the second is element 1.
