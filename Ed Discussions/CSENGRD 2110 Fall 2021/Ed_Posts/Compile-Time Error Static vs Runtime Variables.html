I don't know what a "static value" is. Do you mean the static type of b? At compiletime, only the types of expressions are used; values occur only at run time. So your reasoning is not quite right. The assignment-declaration A a= b; is illegal because for the assignment to work, the object must be cast down to A, and downward casts are not inserted automatically.
