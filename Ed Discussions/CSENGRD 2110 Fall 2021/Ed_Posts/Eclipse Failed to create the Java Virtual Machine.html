Problem: I am getting the message "Failed to create the Java Virtual Machine" Windows Solution 1: If you have Windows, try going to Control Panel. Then search environment variables. Click to edit the system environment variables. Then go to the advance tab. Click Environment Variables near the bottom of that window. Then click "New" under System Variables. Make the variable name "_JAVA_OPTIONS" without the quotation marks, and the value "Xmx256M". If this doesn't work, try 512 instead of 256. Windows Solution 2: Find your eclipse.ini file. Then open it with a text editor. Find -vm in the file and make sure that the path that follows is the correct path to the local java VM you have installed. Mac Solution: Delete the jdk/jre that you downloaded following the original instructions. (Directory: Library/Java/JavaVirtualMachines). Then, you should install the Oracle jdk found here. You will have to create an account with Oracle. Don't worry, it's free. Once you have finished with this, your Eclipse installer should work.
