For that, you should look up method remove in Map in the javadoc API documentation.

Both arguments of the constructor of a Map have to be objects, not elements of a primitive type.

1. if ob is the key of a pair in the Map, the corresponding value is returned, and in this case it is Integer (and not a primitive int value).

2. If ob is NOT the key of a pair in the Map, then null is returned.

Thus, a returned value of null tells you that ob was not a key in the Map.
