This is such a simple thing for you to test yourself, and you would gain so much more by learning the answer yourself.

In ANY class in any project that you have a toString() method, make sure you have @Override before the method. Now, temporarily comment out the body, put in

return 0;

and change the return type in the header to int . Does it compile? If not, what is the error?
