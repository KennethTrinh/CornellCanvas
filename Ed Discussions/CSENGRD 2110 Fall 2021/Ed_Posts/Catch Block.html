Can catch blocks catch exceptions in each other?

In other words, can a catch block in the same try-catch block, catch an exception thrown in the catch block above it?

Author: David Gries (admin)

No. That it not how it works.

If you read carefully how a try-statement is executed, you could answer that question yourself.

Only objects thrown in the try-clause can be caught in a catch block.
