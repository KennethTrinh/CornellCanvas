Author: David Gries (admin)

The expected times to add a value to a hash table and to look for a value in a hash table are both O(1). The expected times for the same operation on a BST is much more --if it is balanced, then O(log n) for a tree with n nodes. We haven't discussed hashing yet, and it is not on prelim 2.
