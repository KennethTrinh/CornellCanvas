When practicing jshell in preparation for lecture 2, Jshell often hung, spinning its wheels after pasting code. We troubleshooted by scouring the internet and found out that some versions of Java 11 have this problem.

Therefore, we installed Java 12 from the website:

https://adoptopenjdk.net/releases.html?variant=openjdk12&jvmVariant=hotspot

choosing (1) OpenJDK 12 and (2) OpenJ9.

Then, jshell worked!

We then made sure that Eclipse compiler used Compiler compliance level 11 on this page:

Preferences -> Java -> Compiler
