No, you cannot use the conditional expression with "?" and ":", as it is essentially an if-statement. (The conditional expression does evaluate a boolean expression, though: please see the JavaHyperText for more details) isOlder() should return the value of "this elephant is older than e." This statement would not be true for elephants of the same age.
