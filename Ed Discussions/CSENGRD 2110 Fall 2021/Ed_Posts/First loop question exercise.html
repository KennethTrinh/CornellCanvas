In the second exercise presented here: https://www.cs.cornell.edu/courses/JavaAndDS/loops/21practiceQ1.pdf The initialization of v=b[n] assumes that the array is of size n+1 at the very least. Wouldn't we want to initialize v as b[0] and define it as the minimum of b[k..n]?

Author: David Gries (admin)

Vote Count: 0

The invariant is. v = min of b[k..n] and 0 <= k <= n Variables v and k are to be initialized. To do this, you want the range k..n to be as small as possible: b[k..n] should contain ONE value, so k has to be n. Then b is b[n]. So the initialization is. k= n; v= b[n];
