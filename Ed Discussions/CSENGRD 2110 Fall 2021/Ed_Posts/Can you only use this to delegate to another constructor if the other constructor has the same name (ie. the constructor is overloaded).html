The question is a bit unclear. Can you try providing an example? But from what I understand, if you are asking if you can have an overloaded constructor that only contains a this statement, then yes. For instance: 
class Book {
    public String title; 

    public Book(String m_title)
    {
        this.title = m_title;
    }
    
    public Book()
    {
        this("Untitled");
    }
}
Using the above example, if one were to create two book objects: Book a = new Book(); and Book b = new Book("Intro to Python") , book 'a' would have a title of "Untitled" and book 'b' would have a title of Intro to Python.
