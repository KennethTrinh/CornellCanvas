Suppose one wrote the first assignment like this: name= name; By the inside-out rule, both references to name refer to parameter name; all this does is get the value in parameter name and store it back in name. You must remember that this evaluated to a pointer to the object in which it appears. Therefore, this.name refers to field name of the object. So this.name= name; evaluates the parameter and stores its value in the field.
