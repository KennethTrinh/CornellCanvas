Author: Troy Moslemi (he/him) (student)

Polling a min heap removes the smallest value, yes. For example from CS2111 lecture, we have an array B of (4,5,8,6,?) Polling array B will return the first value, 4. Polling a max-heap would return the first value in the array, which would also be the largest value in the array.

To be more precise, polling a min-heap removes the value with the smallest priority. If value and priority are synomous, as in some implementations, then it does indeed remove the smallest value
