Data structure is a term that has nothing to do only with Java.

The BST (binary search Tre) is a data structure. It can be implemented in many different languages.

Look at this wikipedia page: https://en.wikipedia.org/wiki/Data_structure

The Collections classes and interfaces, the Collections Framework, is Java's implementation of many data structures.
