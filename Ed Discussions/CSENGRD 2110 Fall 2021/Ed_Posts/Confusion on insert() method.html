Here is the spec, with all the stuff about illegal arguments deleted:

/** Give p a new child, a node with Person c. <br> * Return the newly created node. public H1N1Tree insert(Person p, Person c)

So, yes, you are supposed to create a new H1N1Tree object with Person c in field person.

Then, add this object to field children of the node whose Person is p.
