Author: David Gries (admin)

The Precondition of the second constructor requires that the father parameter has gender 'M'. That implies also that the father parameter must not be null, because null does not point at an object and therefore has no gender. Therefore, an implied Precondition, which you should test for with an assert statement, is that the father parameter is not null.

Same for the mother parameter.
