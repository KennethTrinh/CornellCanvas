Author: Curran Muhlberger (admin)

Vote Count: 1

The superclass, JPanel , already knows how to paint itself, and it will do so by drawing an appropriate background that blends in with the rest of the window (adapting to the current "look-and-feel" settings). You could consider this the "default" look of a component derived from JPanel. When a game is active, we want to start with that default look, then draw on top of it.
