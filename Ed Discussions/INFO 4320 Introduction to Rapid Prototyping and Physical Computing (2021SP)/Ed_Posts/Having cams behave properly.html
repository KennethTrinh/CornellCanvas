Hello, This morning I was asked if one can add gravity to animation so that cams will behave as expected. This is not possible in Fusion 360, but you can get a similar effect as follow: once the rod is in place and its linear constraint is set, select the constrain and set the limit by clicking on the red arrow next to it in the dialog box that opens, set the rest position to the lowest point you want the rod to go (-20mm here). You can do this by typing the value or moving the rod. The final step is to enable contacts: At this point, the rod should follow the cam both upward and downward. Please do not hesitate to let us know if you have any questions about this. Francois.
