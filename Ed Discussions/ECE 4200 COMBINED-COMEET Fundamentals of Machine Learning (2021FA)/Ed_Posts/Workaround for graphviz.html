If you have trouble installing graphviz, you can try the following code instead. 

tree.plot_tree(clf)  #clf is the decision tree you have learned

Replace clf by the name of your decision tree. The output would look something like this: 

Text(22.32, 141.336, 'X[16] <= 0.5\ngini = 0.456\nsamples = 111\nvalue = [39, 72]'),
 Text(8.928, 119.592, 'X[0] <= 0.5\ngini = 0.353\nsamples = 70\nvalue = [16, 54]'),
 Text(4.464, 97.848, 'gini = 0.0\nsamples = 40\nvalue = [0, 40]'),

The first line corresponds to the root node, where you could read the splitting criteria by feature index. Keep in mind that features are indexed from 0 to 17. In this example, X[16] corresponds to the feature with index 16, which is “btm_right_sqr_o”.
