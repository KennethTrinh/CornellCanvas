I'm not quite sure what you mean - In PCA, you find the principal components based on the data covariance matrix ( so the entire dataset is used there ).

If you then want to perform dimensionality reduction, say reducing the dataset to k dimensions, you project each data point onto the first k principal components.

To understand the fraction of variance explained, let us consider a d dimensional dataset which we transform by projecting onto its d principal components. The covariance matrix of the transformed data is, by definition, a diagonal matrix. The sum of the first k diagonal elements of this matrix is therefore the variance explained by the first k components. This sum normalised by the total diagonal sum is what we mean by the fraction of variance explained.
