Author: Jayadev Acharya (admin)

minimizing 1/|w| is the same as maximizing |w|, which is the same as maximizing |w|^2

note |w|^2 is a nice convex function which is now easy to differentiate compared to 1/|w|, which is not convex.

The half is usually put to cancel the square after derivative.
