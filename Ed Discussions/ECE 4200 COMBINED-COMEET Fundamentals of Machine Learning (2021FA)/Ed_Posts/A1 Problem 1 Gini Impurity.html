1. Gini(s) refers to the gini index (as I believe it's called) which you calculate by

gini\left(S\right)\ =\ 1-\sum_{i=1}^l\left(\frac{n_i}{n}\right)^2

2. Gini index is what you use to calculate gini impurity. Gini impurity takes both the set that you're looking at as well as the attribute you are calculating for, so that

gini\left(S;\ A\right)\ =\ \sum_{j=1}^a\left(\frac{\left|S_{v_j}\right|}{\left|S\right|}\cdot gini\left(S_{v_j}\right)\right)

where S is the set and A is the attribute.

3. See number 1.

(presumably) 4. Pick the attribute with the lowest gini impurity value. The closer to 0 it is, the more cases in that node fall into a single category.
