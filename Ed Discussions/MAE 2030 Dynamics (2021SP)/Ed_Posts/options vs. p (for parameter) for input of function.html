p and options are both structs for passing information. p is for getting information, namely numerical constants, into your differential equation file ( myrhs or whatever). Initially we passed p through, say, ODE1, into myrhs. Then we changed that by using anonymous functions. options is for passing information into ODE1 or ODE2 . It is telling the solver what to do. In our case, so far it is telling it how accurate to be, options.RelTol , and the maximum amount of time to try to be that accurate, options.maxtime . Here is a code snippet that should work for the harmonic oscillator 
tspan = ...
z0    = ...
p.k = 1;  p.m = 2;               % set parameters
options.RelTol = 1e-2;           % set desired accuracy
options.maxtime = 10;            % set maximimum amount of try for accuracy
eqn = @(t,z)   myrhs(t,z,p);     % define "anonymous" function eqn
{tarray zarray} = ODE1(eqn, tspan, z0, options);  % call the solver
