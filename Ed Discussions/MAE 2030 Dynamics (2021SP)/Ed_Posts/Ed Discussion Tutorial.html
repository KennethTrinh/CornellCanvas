Here are some tips for effective posts:

Uploading images

  • highly encouraged for explaining work

  • always check to make sure the image is legible after upload

  • when trying to share code, consider using a different option (see "Code snippets" below)

    • Annotating images

  • in-house annotation feature (neat!)

  • will automatically pull up existing images in the thread (no need to download, reupload)

  • resolution might be compromised, so test it out

  • play around with it!

    • Code snippets

  • use the highlighted code insert button (the crossed-out feature isn't interactive)

  • highly encouraged for explaining/debugging code
