The README file can be as simple as a notepad file with clear instructions on how to run your submitted code to get the data/plots you report on your PDF.

For example; if you only submit one plot in your homework and all you have to do to generate that plot is press "Run" on the script editor, then your README can just consist of something along the lines of "Run the included code to generate the plot reported in Page XX of the PDF".
