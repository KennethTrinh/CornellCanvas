    A spring's rest length $L_0$ is it's length when the tension is zero. If $L$ is the present length, then $L_0$ is the length that makes this equations true

    $$ T = k(L - L_0)$$

    The "equilibrium length" is the length of the spring when the whole system is in static equilibrium. For this problem with a hanging weight that would be

    $$L_e = L_0 + mg/k$$

    The purpose of this homework is to get you thinking about those things so that you are taking a step from

    $$\ddot x + x = f$$

    towards dealing with real systems. Read about springs in Section 7.1 of the textbook.
