Author: Andy Ruina (admin)

As a rule, problems should be distinct so that they can be read independently. But if you have a file for, say, myEulerSolver.m, you need not copy that file again and again. Somehow, however, anyone should be able to read and understand a single problem on its own. Imagine another student trying to learn just one problem by looking at your homework. You don't want to make that a puzzle or a complex operation.
