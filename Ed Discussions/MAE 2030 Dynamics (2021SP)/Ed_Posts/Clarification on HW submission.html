Answer: a single PDF with everything as well as separate .m files More information here: https://canvas.cornell.edu/courses/25081/pages/homework-information For each homework assignment, students will submit the following: A single .pdf file, containing the following: Written work. MATLAB code. Plots and figures. MATLAB code also as .m file(s). If more than one, place in .zip folder, but leave .pdf outside. Include a README file explaining how to run the code.
